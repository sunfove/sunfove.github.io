插入音乐

方案一

简介
利用html中的audio 标签插入音乐。 audio标签定义声音，比如音乐或其他音频流。

案例
优缺点
优点：音乐可以自己选择，不受曲库限制

缺点：对音乐格式有要求，目前audio 元素支持的3种文件格式：MP3、Wav、Ogg

方案二

简介
利用网易云音乐生成外链播放器后分享。 现在网易云音乐官网找到自己想要的音乐，

点击音乐下方的“生成外链播放器”，然后进入外链生成界面，复制最下方的HTML代码，拷贝到自己想放的地方即可。这里边框的大小可以自由调节。

案例
优缺点
优点：网易云直接播放，不用自己存储调用

缺点：很多音乐有版权保护，无法生成外链
