<!--more-->

### 插入图片

#### 方案一 
<center style='color: red'><b>简介</b></center>
![]()是markdown中常用的引用图片的格式，[]中是图片说明，()中写图片路径或者url。
```
![图片说明]('url或图片路径')
```
<center style='color: red'><b>案例</b></center>
```
![方案一](https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/Hexo/S.png)
```

![方案一](https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/Hexo/S.png)

#### 方案二
<center style='color: red'><b>简介</b></center>
```
<img src='url或图片路径' height=400px'>
```
img标签定义 HTML 页面中的图像，img可修改的属性有

![](https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/Hexo/Hexo%E4%B8%AD%E6%8F%92%E5%85%A5%E5%9B%BE%E7%89%87%E5%92%8C%E9%9F%B3%E4%B9%90%E7%9A%84%E5%87%A0%E7%A7%8D%E6%96%B9%E5%BC%8F4.png)
<center style='color: red'><b>案例</b></center>
```
<img src='https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/Hexo/S.png' height=400px >
```
<img src='https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/Hexo/S.png' height=400px >

#### 方案三
<center style='color: red'><b>简介</b></center>
```
{% asset_img 1.jpg 这是一个新的博客的图片的说明 %}
```
方案三需要现在hexo配置文件中将post_asset_folder打开，这样hexo新建博客的时候，会生成相同名称的文件夹，将图片放到文件夹中，可以通过其相对位置进行直接引用。
```
post_asset_folder: true
```
<center style='color: red'><b>案例</b></center>
```
{% asset_img S.png 方案三 %}
```
{% asset_img S.png 方案三 %}



### 插入音乐

#### 方案一 
<center style='color: red'><b>简介</b></center>
利用html中的audio 标签插入音乐。
```
<audio autoplay>
  <source src=" url 或 文件路径" type="audio/mpeg">
</audio>
```
audio标签定义声音，比如音乐或其他音频流。

![](https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/Hexo/Hexo%E4%B8%AD%E6%8F%92%E5%85%A5%E5%9B%BE%E7%89%87%E5%92%8C%E9%9F%B3%E4%B9%90%E7%9A%84%E5%87%A0%E7%A7%8D%E6%96%B9%E5%BC%8F1.png)
<center style='color: red'><b>案例</b></center>
```
<audio controls>
  <source src="https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/%E9%9F%B3%E4%B9%90/Hold%20back%20the%20river.mp3" type="audio/mpeg">
</audio>
```
<audio controls>
  <source src="https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/%E9%9F%B3%E4%B9%90/Hold%20back%20the%20river.mp3" type="audio/mpeg">
</audio>
<center style='color: red'><b>优缺点</b></center>
优点：音乐可以自己选择，不受曲库限制

缺点：对音乐格式有要求，目前audio 元素支持的3种文件格式：MP3、Wav、Ogg

#### 方案二
<center style='color: red'><b>简介</b></center>
利用网易云音乐生成外链播放器后分享。
现在网易云音乐官网找到自己想要的音乐，

![](https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/Hexo/Hexo%E4%B8%AD%E6%8F%92%E5%85%A5%E5%9B%BE%E7%89%87%E5%92%8C%E9%9F%B3%E4%B9%90%E7%9A%84%E5%87%A0%E7%A7%8D%E6%96%B9%E5%BC%8F3.png)

点击音乐下方的“生成外链播放器”，然后进入外链生成界面，复制最下方的HTML代码，拷贝到自己想放的地方即可。这里边框的大小可以自由调节。

![](https://suncos-01-1254144885.cos.ap-shanghai.myqcloud.com/Hexo/Hexo%E4%B8%AD%E6%8F%92%E5%85%A5%E5%9B%BE%E7%89%87%E5%92%8C%E9%9F%B3%E4%B9%90%E7%9A%84%E5%87%A0%E7%A7%8D%E6%96%B9%E5%BC%8F2.png)
<center style='color: red'><b>案例</b></center>
```
<iframe frameborder="no" border="0" marginwidth="0" marginheight="0" width=330 height=86 src="//music.163.com/outchain/player?type=2&id=26090155&auto=0&height=66"></iframe>
```
<iframe frameborder="no" border="0" marginwidth="0" marginheight="0" width=330 height=86 src="//music.163.com/outchain/player?type=2&id=26090155&auto=0&height=66"></iframe>
<center style='color: red'><b>优缺点</b></center>
优点：网易云直接播放，不用自己存储调用

缺点：很多音乐有版权保护，无法生成外链
