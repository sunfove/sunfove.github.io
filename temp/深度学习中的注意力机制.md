## 引言

在自然语言处理领域，2017年是一个分水岭。Google团队发表的论文《Attention Is All You Need》提出了Transformer架构，它完全抛弃了传统的循环神经网络（RNN）和卷积神经网络（CNN），仅依靠**注意力机制**就实现了当时最优的机器翻译性能。

这一创新不仅改变了NLP领域，更成为GPT、BERT、ChatGPT等大语言模型的基石。理解注意力机制，是理解现代AI的核心。

---

## 01. 直觉理解：人类如何阅读

当我们阅读句子"**The animal didn't cross the street because it was too tired**"时，为了理解"it"指代什么，我们的大脑会自然地回顾"animal"和"street"这两个词，并判断"it"更可能指代"animal"（因为疲惫的是动物）。

**注意力机制的核心思想**：让模型学会"看向哪里"——根据上下文动态地为每个词分配不同的权重。

---

## 02. 从编码器-解码器说起

### 传统Seq2Seq的问题

传统的序列到序列（Seq2Seq）模型使用RNN：

```
输入序列 → [编码器RNN] → 固定长度上下文向量 → [解码器RNN] → 输出序列
```

**瓶颈**：所有输入信息被压缩到一个固定长度的向量中。当输入序列很长时，信息会丢失。

### 注意力的引入

注意力机制允许解码器在每一步生成输出时，**动态地关注编码器的不同位置**：

$$
c_i = \sum_{j=1}^{T_x} \alpha_{ij} h_j
$$

- $c_i$：第 $i$ 步的上下文向量
- $h_j$：编码器第 $j$ 步的隐状态
- $\alpha_{ij}$：注意力权重（解码器第 $i$ 步对编码器第 $j$ 步的关注程度）

---

## 03. Self-Attention：自己关注自己

Transformer的核心创新是**自注意力（Self-Attention）**：序列中的每个元素都与序列中的其他所有元素计算相关性。

### Query, Key, Value

借鉴信息检索的概念，Self-Attention引入三个向量：

| 概念 | 含义 | 类比 |
|------|------|------|
| **Query (Q)** | 当前元素的查询 | 我要找什么 |
| **Key (K)** | 其他元素的键 | 别人的标签 |
| **Value (V)** | 其他元素的值 | 别人的内容 |

对于输入序列 $X = [x_1, x_2, ..., x_n]$，通过三个线性变换得到：

$$Q = XW^Q, \quad K = XW^K, \quad V = XW^V$$

其中 $W^Q, W^K, W^V$ 是可学习的参数矩阵。

### Scaled Dot-Product Attention

注意力权重的计算公式：

$$
\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right) V
$$

**分步解析**：

1. **相似度计算**：$QK^T$ 得到 $n \times n$ 的注意力分数矩阵
2. **缩放**：除以 $\sqrt{d_k}$ 防止梯度消失（$d_k$ 是Key的维度）
3. **归一化**：softmax 使每行权重和为1
4. **加权求和**：用权重对 Value 进行加权

### 为什么除以 $\sqrt{d_k}$？

当 $d_k$ 较大时，$QK^T$ 的元素值会很大，导致 softmax 进入饱和区，梯度接近0。

假设 $q$ 和 $k$ 的元素独立同分布，均值为0，方差为1：

$$
\text{Var}(q \cdot k) = d_k \cdot \text{Var}(q_i k_i) = d_k
$$

除以 $\sqrt{d_k}$ 后，方差变为1，保持数值稳定。

## 04. Multi-Head Attention

单次注意力可能只关注一种类型的关系。**多头注意力**让模型从多个角度观察：

$$
\text{MultiHead}(Q,K,V) = \text{Concat}(\text{head}_1, ..., \text{head}_h) W^O
$$

其中：

$$
\text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)
$$

- $h$：头的数量（通常为8）
- 每个"头"关注不同的语义关系
- $W_i^Q \in \mathbb{R}^{d_{model} \times d_k}$，$d_k = d_{model}/h$

```
输入: [batch, seq_len, d_model]
         ↓
    ┌────┬────┬────┬────┐
    │Head1│Head2│...│Head8│  (并行计算8个注意力)
    └────┴────┴────┴────┘
         ↓ Concat
    [batch, seq_len, d_model]
         ↓ 线性变换 W^O
输出: [batch, seq_len, d_model]
```

---

## 05. 完整的Transformer架构

### 编码器（Encoder）

每个编码器层包含两个子层：

1. **多头自注意力** + 残差连接 + LayerNorm
2. **前馈神经网络** + 残差连接 + LayerNorm

$$
\text{FFN}(x) = \max(0, xW_1 + b_1)W_2 + b_2
$$

### 解码器（Decoder）

解码器额外包含：

3. **掩码多头注意力**（防止看到未来信息）

### 位置编码

由于Self-Attention没有序列顺序概念，需要添加位置编码：

$$
PE_{(pos, 2i)} = \sin(pos / 10000^{2i/d_{model}})
$$
$$
PE_{(pos, 2i+1)} = \cos(pos / 10000^{2i/d_{model}})
$$

## 06. PyTorch实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

class SelfAttention(nn.Module):
    """单头自注意力"""
    def __init__(self, d_model):
        super().__init__()
        self.d_k = d_model
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)
        
    def forward(self, x, mask=None):
        # x: [batch, seq_len, d_model]
        Q = self.W_q(x)  # [batch, seq_len, d_model]
        K = self.W_k(x)
        V = self.W_v(x)
        
        # 计算注意力分数
        scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(self.d_k)
        
        # 应用掩码（用于解码器）
        if mask is not None:
            scores = scores.masked_fill(mask == 0, -1e9)
        
        # Softmax归一化
        attn_weights = F.softmax(scores, dim=-1)
        
        # 加权求和
        output = torch.matmul(attn_weights, V)
        return output, attn_weights


class MultiHeadAttention(nn.Module):
    """多头注意力"""
    def __init__(self, d_model, num_heads=8):
        super().__init__()
        assert d_model % num_heads == 0
        
        self.d_k = d_model // num_heads
        self.num_heads = num_heads
        
        self.W_q = nn.Linear(d_model, d_model)
        self.W_k = nn.Linear(d_model, d_model)
        self.W_v = nn.Linear(d_model, d_model)
        self.W_o = nn.Linear(d_model, d_model)
        
    def forward(self, x, mask=None):
        batch_size, seq_len, d_model = x.shape
        
        # 线性变换
        Q = self.W_q(x).view(batch_size, seq_len, self.num_heads, self.d_k).transpose(1, 2)
        K = self.W_k(x).view(batch_size, seq_len, self.num_heads, self.d_k).transpose(1, 2)
        V = self.W_v(x).view(batch_size, seq_len, self.num_heads, self.d_k).transpose(1, 2)
        # Q, K, V: [batch, num_heads, seq_len, d_k]
        
        # 注意力计算
        scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(self.d_k)
        
        if mask is not None:
            mask = mask.unsqueeze(1)  # 扩展head维度
            scores = scores.masked_fill(mask == 0, -1e9)
        
        attn_weights = F.softmax(scores, dim=-1)
        attn_output = torch.matmul(attn_weights, V)
        
        # 拼接多头
        attn_output = attn_output.transpose(1, 2).contiguous().view(batch_size, seq_len, d_model)
        
        # 最终线性变换
        output = self.W_o(attn_output)
        return output, attn_weights


class TransformerEncoderLayer(nn.Module):
    """Transformer编码器层"""
    def __init__(self, d_model, num_heads, d_ff, dropout=0.1):
        super().__init__()
        self.self_attn = MultiHeadAttention(d_model, num_heads)
        self.ffn = nn.Sequential(
            nn.Linear(d_model, d_ff),
            nn.ReLU(),
            nn.Linear(d_ff, d_model)
        )
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        self.dropout = nn.Dropout(dropout)
        
    def forward(self, x, mask=None):
        # 自注意力 + 残差连接 + LayerNorm
        attn_output, _ = self.self_attn(x, mask)
        x = self.norm1(x + self.dropout(attn_output))
        
        # 前馈网络 + 残差连接 + LayerNorm
        ffn_output = self.ffn(x)
        x = self.norm2(x + self.dropout(ffn_output))
        
        return x


# 测试
if __name__ == "__main__":
    d_model = 512
    num_heads = 8
    batch_size = 2
    seq_len = 10
    
    # 创建测试输入
    x = torch.randn(batch_size, seq_len, d_model)
    
    # 单头注意力
    self_attn = SelfAttention(d_model)
    output, weights = self_attn(x)
    print(f"单头注意力输出形状: {output.shape}")
    print(f"注意力权重形状: {weights.shape}")
    
    # 多头注意力
    multi_attn = MultiHeadAttention(d_model, num_heads)
    output, weights = multi_attn(x)
    print(f"多头注意力输出形状: {output.shape}")
    
    # Transformer编码器层
    encoder_layer = TransformerEncoderLayer(d_model, num_heads, d_ff=2048)
    output = encoder_layer(x)
    print(f"编码器层输出形状: {output.shape}")
```

多头注意力输出形状: torch.Size([2, 10, 512])

## 07. 可视化注意力权重

```python
import matplotlib.pyplot as plt
import seaborn as sns

def visualize_attention(attn_weights, tokens=None, title="Attention Weights"):
    """可视化注意力权重"""
    plt.figure(figsize=(8, 6))
    
    # 取第一个样本的平均注意力
    weights = attn_weights[0].mean(dim=0).detach().numpy()
    
    sns.heatmap(weights, cmap='Blues', annot=True, fmt='.2f',
                xticklabels=tokens, yticklabels=tokens)
    plt.title(title)
    plt.xlabel('Key')
    plt.ylabel('Query')
    plt.tight_layout()
    plt.savefig('temp/attention_weights.png', dpi=150, bbox_inches='tight')
    plt.show()

# 测试可视化
tokens = ['The', 'cat', 'sat', 'on', 'the', 'mat']
d_model = 64
x = torch.randn(1, len(tokens), d_model)

attn = SelfAttention(d_model)
output, weights = attn(x)
visualize_attention(weights, tokens, "Self-Attention Weights")
print(f"注意力权重可视化已保存")
```

## 08. 复杂度分析

| 模型 | 时间复杂度 | 空间复杂度 | 并行性 |
|------|-----------|-----------|--------|
| RNN | $O(n \cdot d^2)$ | $O(d)$ | 串行 |
| CNN | $O(k \cdot n \cdot d^2)$ | $O(k \cdot d)$ | 有限 |
| **Self-Attention** | $O(n^2 \cdot d)$ | $O(n^2)$ | 完全并行 |

- $n$：序列长度
- $d$：模型维度
- $k$：卷积核大小

**Self-Attention的优势**：
- 完全并行计算，训练速度快
- 能够捕捉长距离依赖
- 每层都能看到整个序列

**Self-Attention的劣势**：
- $O(n^2)$ 复杂度，对超长序列不友好
- 这也是为什么GPT有上下文长度限制

## 总结

| 核心概念 | 要点 |
|---------|------|
| **Self-Attention** | $Attention(Q,K,V) = softmax(QK^T/\sqrt{d_k})V$ |
| **Multi-Head** | 多角度并行计算，捕捉不同语义关系 |
| **Q/K/V** | Query（查询）、Key（键）、Value（值）的信息检索类比 |
| **缩放因子** | $\sqrt{d_k}$ 防止softmax梯度消失 |
| **残差连接** | 缓解深层网络训练问题 |

Transformer的成功证明了：**注意力机制是一种强大的、通用的序列建模方式**。从GPT到BERT，从视觉Transformer到多模态大模型，注意力机制已经成为现代深度学习的核心组件。
