# 遗传算法与进化策略：从自然选择到算法优化

## 引言

在人工智能和优化领域，我们常常面临这样的困境：当问题空间复杂、非线性、多模态时，传统的梯度下降法或穷举搜索往往力不从心。想象一下，你需要设计一个航空发动机的叶片形状，或者优化一个包含数百个参数的神经网络结构——这些问题的解空间如此庞大，以至于任何确定性算法都会陷入局部最优或计算时间爆炸。

正是在这样的背景下，遗传算法（Genetic Algorithm, GA）和进化策略（Evolutionary Strategy, ES）应运而生。它们借鉴了达尔文自然选择的核心思想——“适者生存”——通过模拟种群的进化过程来搜索最优解。自1975年John Holland正式提出遗传算法以来，这类进化算法已成为解决复杂优化问题的利器，在工程、金融、生物信息学等领域展现出惊人的效果。

本文将带你深入理解遗传算法的核心概念、数学原理，并通过完整的Python代码实现一个可运行的遗传算法。无论你是算法爱好者还是实践工程师，都能从中获得启发。

## 核心概念

### 种群（Population）

在遗传算法中，**种群**是一组候选解的集合。每个候选解被称为一个**个体**（Individual），而个体通常由**染色体**（Chromosome）表示。染色体由一系列**基因**（Gene）组成，基因可以是二进制位、实数或其他数据结构。

例如，对于优化函数 f(x) = x²，其中 x 的取值范围是 [0, 31]，我们可以用5位二进制数表示染色体：`01011` 对应十进制 11。种群大小（Population Size）通常设为 20-100，过小会导致早熟收敛，过大会增加计算成本。

### 适应度函数（Fitness Function）

**适应度函数**用于评估每个个体的优劣程度。在优化问题中，它通常直接对应目标函数（对于最大化问题）或目标函数的倒数/负值（对于最小化问题）。

数学上，适应度函数定义为：
$$
F: \text{染色体空间} \to \mathbb{R}^+
$$

例如，对于最大化问题 f(x) = x²，适应度函数可设为：
$$
F(\text{染色体}) = f(\text{解码}(染色体))
$$

适应度值越高，个体被选中繁殖的概率越大。

### 选择（Selection）

**选择**操作模拟了自然选择中的“优胜劣汰”，它从当前种群中挑选出适应度较高的个体作为父代，用于产生下一代。

最经典的选择方法是**轮盘赌选择**（Roulette Wheel Selection）。每个个体被选中的概率与其适应度成正比：
$$
P_i = \frac{F_i}{\sum_{j=1}^{N} F_j}
$$
其中，N 是种群大小，F_i 是第 i 个个体的适应度。

## 数学原理

### 轮盘赌选择概率

轮盘赌选择的数学本质是概率采样。假设我们有四个个体，适应度分别为 [10, 20, 30, 40]，总适应度 sum = 100。则每个个体被选中的概率为：
$$
P = [0.1, 0.2, 0.3, 0.4]
$$

算法实现时，我们生成一个 [0, 1) 的随机数 r，然后累加概率直到超过 r，对应的个体即为选中个体。这个过程可以用累积分布函数（CDF）来描述：
$$
\text{CDF}(i) = \sum_{j=1}^{i} P_j
$$
当 r ∈ [CDF(i-1), CDF(i)) 时，选中第 i 个个体。

### 单点交叉（Single-Point Crossover）

**交叉**操作模拟了生物繁殖中的基因重组。单点交叉是最简单的交叉方式：在两个父代染色体的随机位置选择一个切点，交换切点后的基因片段，产生两个子代。

假设父代染色体为：
- 父代1: `1010|1101`
- 父代2: `0111|0010`

切点选在第4位后（|位置），则子代为：
- 子代1: `1010|0010`
- 子代2: `0111|1101`

交叉操作的概率由**交叉率**（Crossover Rate, P_c）控制，通常设为 0.6-0.9。交叉率越高，种群多样性越强，但可能破坏优秀基因组合。

### 变异（Mutation）

为了保持种群多样性并防止早熟收敛，还需要**变异**操作。对于二进制编码，变异以很小的概率（通常 0.001-0.01）翻转基因位：
$$
P(\text{基因翻转}) = P_m
$$

## 代码实现

下面是一个完整的Python实现，用于求解函数 f(x) = x² 在 [0, 31] 区间上的最大值：

```python
import random
import math

# 遗传算法参数
POP_SIZE = 20          # 种群大小
CHROM_LEN = 5          # 染色体长度（二进制位）
GENERATIONS = 50       # 进化代数
CROSSOVER_RATE = 0.8   # 交叉率
MUTATION_RATE = 0.01   # 变异率

def decode(chromosome):
    """将二进制染色体解码为十进制数"""
    return int(''.join(str(bit) for bit in chromosome), 2)

def fitness(x):
    """适应度函数：求解 f(x) = x^2 的最大值"""
    return x ** 2

def initialize_population():
    """初始化种群：随机生成二进制染色体"""
    population = []
    for _ in range(POP_SIZE):
        chromosome = [random.randint(0, 1) for _ in range(CHROM_LEN)]
        population.append(chromosome)
    return population

def evaluate_population(population):
    """评估种群中每个个体的适应度"""
    fitness_values = []
    for chrom in population:
        x = decode(chrom)
        fitness_values.append(fitness(x))
    return fitness_values

def selection(population, fitness_values):
    """轮盘赌选择：根据适应度概率选择父代"""
    total_fitness = sum(fitness_values)
    probabilities = [f / total_fitness for f in fitness_values]
    
    # 累积分布
    cumulative_prob = []
    cum_sum = 0
    for p in probabilities:
        cum_sum += p
        cumulative_prob.append(cum_sum)
    
    # 选择两个父代
    selected = []
    for _ in range(2):
        r = random.random()
        for i, cp in enumerate(cumulative_prob):
            if r <= cp:
                selected.append(population[i])
                break
    return selected[0], selected[1]

def crossover(parent1, parent2):
    """单点交叉"""
    if random.random() < CROSSOVER_RATE:
        # 随机选择切点（1到CHROM_LEN-1之间）
        crossover_point = random.randint(1, CHROM_LEN - 1)
        child1 = parent1[:crossover_point] + parent2[crossover_point:]
        child2 = parent2[:crossover_point] + parent1[crossover_point:]
        return child1, child2
    else:
        return parent1[:], parent2[:]

def mutation(chromosome):
    """变异：以MUTATION_RATE的概率翻转每个基因"""
    mutated = []
    for gene in chromosome:
        if random.random() < MUTATION_RATE:
            mutated.append(1 - gene)  # 翻转
        else:
            mutated.append(gene)
    return mutated

def genetic_algorithm():
    """主算法流程"""
    population = initialize_population()
    
    for gen in range(GENERATIONS):
        # 评估当前种群
        fitness_values = evaluate_population(population)
        best_idx = fitness_values.index(max(fitness_values))
        best_x = decode(population[best_idx])
        best_fitness = fitness_values[best_idx]
        
        print(f"Generation {gen}: best x = {best_x}, fitness = {best_fitness}")
        
        # 生成新一代种群
        new_population = []
        
        # 精英保留：保留当前最优个体
        new_population.append(population[best_idx][:])
        
        while len(new_population) < POP_SIZE:
            # 选择父代
            parent1, parent2 = selection(population, fitness_values)
            # 交叉
            child1, child2 = crossover(parent1, parent2)
            # 变异
            child1 = mutation(child1)
            child2 = mutation(child2)
            # 加入新种群
            new_population.append(child1)
            if len(new_population) < POP_SIZE:
                new_population.append(child2)
        
        population = new_population
    
    # 最终结果
    fitness_values = evaluate_population(population)
    best_idx = fitness_values.index(max(fitness_values))
    best_x = decode(population[best_idx])
    best_fitness = fitness_values[best_idx]
    print(f"\nFinal result: x = {best_x}, f(x) = {best_fitness}")
    return best_x, best_fitness

if __name__ == "__main__":
    best_x, best_fitness = genetic_algorithm()
```

运行这段代码，你会看到种群逐渐收敛到最优解 x=31（二进制11111），f(x)=961。

## 应用场景

### 1. 旅行商问题（TSP）

TSP是经典的NP难问题：给定一组城市和城市间的距离，求一条经过所有城市且总距离最短的路径。遗传算法在这里大显身手：

- **编码**：用城市序号排列表示路径，如 [3, 1, 4, 2, 5]
- **适应度**：路径总距离的倒数
- **特殊操作**：由于路径不能重复，交叉和变异需要采用顺序交叉（OX）或交换变异等特殊算子

### 2. 神经网络结构优化

在深度学习领域，设计网络结构（层数、每层神经元数、激活函数等）往往依赖经验。遗传算法可以自动化这个过程：

- **编码**：用固定长度的向量表示网络超参数，如 [3, 128, 64, 0.5] 表示3个全连接层，128和64个神经元，dropout率0.5
- **适应度**：在验证集上的准确率
- **结果**：GA能在几十代内找到比手动调参更好的网络结构

## 总结

遗传算法的核心思想可以概括为三个关键词：**编码、适应度、进化操作**。通过本文，我们深入理解了：

1. **种群与编码**：将问题解空间映射到染色体空间，这是算法的基础
2. **适应度函数**：连接问题目标与算法搜索的桥梁
3. **选择-交叉-变异**：三种操作分别负责保留优秀基因、探索新组合、维持多样性

从数学上看，遗传算法是一种随机搜索算法，它在概率意义上保证了收敛到全局最优（当满足某些条件时）。虽然对于简单问题，它可能不如梯度下降高效，但在面对复杂、非凸、离散的优化问题时，它往往能给出令人惊喜的结果。

进化计算的世界远不止于此：进化策略（ES）使用实数编码和自适应变异步长，遗传编程（GP）进化出计算机程序，多目标进化算法（如NSGA-II）同时优化多个目标。无论你选择哪个方向，理解遗传算法的基本原理都是最好的起点。
