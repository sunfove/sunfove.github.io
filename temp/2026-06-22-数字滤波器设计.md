# 数字滤波器设计：从数学原理到Python实现

## 引言

在数字信号处理的世界里，滤波器无处不在。从智能手机降噪到心电图信号去噪，从音频均衡器到雷达信号处理，数字滤波器是信号处理系统的核心组件。但什么是数字滤波器？简单来说，它是一种数学算法，能够有选择地增强或抑制信号中的特定频率成分。本文将深入探讨数字滤波器的数学基础——线性时不变系统（LTI），介绍两种主要类型——有限脉冲响应（FIR）和无限脉冲响应（IIR）滤波器，并通过完整的Python代码展示如何从零设计和实现它们。

## 核心概念

### 线性时不变系统（LTI）

LTI系统是信号处理理论的基石。一个系统如果同时满足**线性性**和**时不变性**，就被称为LTI系统。

- **线性性**：系统对输入的加权和等于输出的加权和。即如果输入 \(x_1(t)\) 产生输出 \(y_1(t)\)，输入 \(x_2(t)\) 产生输出 \(y_2(t)\)，那么输入 \(a x_1(t) + b x_2(t)\) 产生输出 \(a y_1(t) + b y_2(t)\)。

- **时不变性**：如果输入延迟一段时间，输出也相应延迟相同的时间。即如果输入 \(x(t)\) 产生输出 \(y(t)\)，那么输入 \(x(t-\tau)\) 产生输出 \(y(t-\tau)\)。

这两个性质使得LTI系统可以用一个单一的冲激响应 \(h[n]\) 完全描述。

### 频率响应

频率响应描述了系统对不同频率正弦信号的增益和相移。对于LTI系统，频率响应 \(H(e^{j\omega})\) 是冲激响应的离散时间傅里叶变换（DTFT）：

$$H(e^{j\omega}) = \sum_{n=-\infty}^{\infty} h[n] e^{-j\omega n}$$

其中 \(\omega\) 是归一化角频率，范围从 \(0\) 到 \(2\pi\)。频率响应的幅度 \(|H(e^{j\omega})|\) 决定了系统对不同频率成分的放大或衰减程度。

### FIR与IIR滤波器

**FIR滤波器**的冲激响应是有限长的，即只有有限个非零值。其输出只依赖于当前和过去的输入样本：

$$y[n] = \sum_{k=0}^{M} b_k x[n-k]$$

**IIR滤波器**的冲激响应是无限长的，输出不仅依赖于输入，还依赖于过去的输出：

$$y[n] = \sum_{k=0}^{M} b_k x[n-k] - \sum_{k=1}^{N} a_k y[n-k]$$

## 数学原理

### 卷积和

LTI系统的输出可以通过输入与冲激响应的卷积得到：

$$y[n] = x[n] * h[n] = \sum_{k=-\infty}^{\infty} x[k] h[n-k]$$

对于因果系统（\(h[n]=0\) 当 \(n<0\)），求和范围变为 \(k=0\) 到 \(n\)。

### 传递函数

对差分方程应用Z变换，可以得到系统的传递函数。

对于FIR滤波器：

$$H(z) = \sum_{k=0}^{M} b_k z^{-k}$$

对于IIR滤波器：

$$H(z) = \frac{\sum_{k=0}^{M} b_k z^{-k}}{1 + \sum_{k=1}^{N} a_k z^{-k}}$$

传递函数的极点和零点位置决定了滤波器的频率响应特性。极点靠近单位圆会导致频率响应在该频率处出现峰值，零点靠近单位圆则会导致陷波。

### 设计方法

**窗函数法**（用于FIR）：首先设计理想滤波器的冲激响应 \(h_d[n]\)，然后乘以窗函数 \(w[n]\) 得到有限长的冲激响应：

$$h[n] = h_d[n] \cdot w[n]$$

常用的窗函数包括汉明窗、海明窗、布莱克曼窗等。

**双线性变换法**（用于IIR）：将模拟滤波器的s平面映射到数字滤波器的z平面：

$$s = \frac{2}{T} \frac{1 - z^{-1}}{1 + z^{-1}}$$

这种方法避免了频率混叠，但会引入频率扭曲，可以通过预扭曲补偿。

## 代码实现

下面是一个完整的Python实现，展示如何设计和应用FIR和IIR低通滤波器：

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import signal
from scipy.fft import fft, fftfreq

def design_fir_lowpass(cutoff_freq, num_taps, fs=1000):
    """
    使用窗函数法设计FIR低通滤波器
    
    参数:
        cutoff_freq: 截止频率 (Hz)
        num_taps: 滤波器阶数+1 (必须为奇数)
        fs: 采样频率 (Hz)
    
    返回:
        fir_coeffs: FIR滤波器系数
    """
    # 归一化截止频率
    nyquist = fs / 2
    normal_cutoff = cutoff_freq / nyquist
    
    # 设计理想低通滤波器的冲激响应
    taps = np.zeros(num_taps)
    mid = num_taps // 2
    
    for i in range(num_taps):
        if i == mid:
            taps[i] = 2 * normal_cutoff
        else:
            n = i - mid
            taps[i] = np.sin(2 * np.pi * normal_cutoff * n) / (np.pi * n)
    
    # 应用汉明窗
    window = np.hamming(num_taps)
    fir_coeffs = taps * window
    
    # 归一化
    fir_coeffs = fir_coeffs / np.sum(fir_coeffs)
    
    return fir_coeffs

def design_iir_lowpass(cutoff_freq, order, fs=1000, ripple=0.5, stopband_atten=40):
    """
    使用双线性变换法设计IIR低通滤波器（巴特沃斯型）
    
    参数:
        cutoff_freq: 截止频率 (Hz)
        order: 滤波器阶数
        fs: 采样频率 (Hz)
        ripple: 通带纹波 (dB)
        stopband_atten: 阻带衰减 (dB)
    
    返回:
        b, a: IIR滤波器系数
    """
    nyquist = fs / 2
    normal_cutoff = cutoff_freq / nyquist
    
    # 设计巴特沃斯滤波器
    b, a = signal.butter(order, normal_cutoff, btype='low', analog=False)
    
    return b, a

def apply_filter(signal_data, filter_coeffs, filter_type='fir', a_coeffs=None):
    """
    应用滤波器到信号
    
    参数:
        signal_data: 输入信号
        filter_coeffs: FIR系数或IIR的b系数
        filter_type: 'fir' 或 'iir'
        a_coeffs: IIR的a系数（仅IIR需要）
    
    返回:
        filtered_signal: 滤波后的信号
    """
    if filter_type == 'fir':
        # FIR滤波
        filtered_signal = np.convolve(signal_data, filter_coeffs, mode='same')
    elif filter_type == 'iir':
        # IIR滤波
        filtered_signal = signal.lfilter(filter_coeffs, a_coeffs, signal_data)
    
    return filtered_signal

# 示例：生成测试信号并应用滤波器
fs = 1000  # 采样频率
t = np.linspace(0, 1, fs, endpoint=False)

# 创建包含50Hz和200Hz成分的测试信号
freq1, freq2 = 50, 200
signal_test = np.sin(2 * np.pi * freq1 * t) + 0.5 * np.sin(2 * np.pi * freq2 * t)

# 设计FIR低通滤波器（截止频率100Hz）
fir_coeffs = design_fir_lowpass(cutoff_freq=100, num_taps=51, fs=fs)

# 设计IIR低通滤波器（截止频率100Hz）
iir_b, iir_a = design_iir_lowpass(cutoff_freq=100, order=4, fs=fs)

# 应用滤波器
fir_filtered = apply_filter(signal_test, fir_coeffs, filter_type='fir')
iir_filtered = apply_filter(signal_test, iir_b, filter_type='iir', a_coeffs=iir_a)

# 绘制结果
plt.figure(figsize=(12, 8))

plt.subplot(3, 1, 1)
plt.plot(t, signal_test)
plt.title('原始信号（50Hz + 200Hz）')
plt.xlabel('时间 (s)')
plt.ylabel('幅度')

plt.subplot(3, 1, 2)
plt.plot(t, fir_filtered)
plt.title('FIR滤波后信号（截止频率100Hz）')
plt.xlabel('时间 (s)')
plt.ylabel('幅度')

plt.subplot(3, 1, 3)
plt.plot(t, iir_filtered)
plt.title('IIR滤波后信号（截止频率100Hz）')
plt.xlabel('时间 (s)')
plt.ylabel('幅度')

plt.tight_layout()
plt.show()

# 计算并绘制频率响应
plt.figure(figsize=(10, 6))

# FIR频率响应
w_fir, h_fir = signal.freqz(fir_coeffs, worN=8000)
plt.semilogx(w_fir * fs / (2 * np.pi), 20 * np.log10(abs(h_fir)), label='FIR')

# IIR频率响应
w_iir, h_iir = signal.freqz(iir_b, iir_a, worN=8000)
plt.semilogx(w_iir * fs / (2 * np.pi), 20 * np.log10(abs(h_iir)), label='IIR')

plt.axvline(100, color='red', linestyle='--', label='截止频率')
plt.grid(True, which='both', linestyle='--', alpha=0.6)
plt.xlabel('频率 (Hz)')
plt.ylabel('幅度 (dB)')
plt.title('滤波器频率响应')
plt.legend()
plt.ylim(-60, 5)
plt.show()
```

## 应用场景

### 1. 心电图（ECG）信号去噪

心电图信号包含重要的生理信息，但容易受到工频干扰（50/60Hz）和肌电噪声的影响。使用带通滤波器（0.5-40Hz）可以保留心电信号的主要频率成分，同时抑制噪声。FIR滤波器因其线性相位特性，在医学应用中特别受欢迎，因为它不会扭曲信号的时域特征。

### 2. 音频均衡器

现代音频处理系统广泛使用数字滤波器实现均衡功能。例如，使用多个带通IIR滤波器（如参量均衡器）可以独立调整不同频段的增益。IIR滤波器因其计算效率高（低阶数即可实现陡峭的过渡带），在实时音频处理中具有明显优势。典型的应用包括：
- 低音增强（低频提升）
- 齿音消除（高频衰减）
- 房间声学校正

## 总结

本文深入探讨了数字滤波器的核心概念和数学原理，并通过完整的Python代码展示了如何设计和实现FIR与IIR滤波器。关键要点回顾：

1. **LTI系统**是数字滤波器的理论基础，其冲激响应完全描述了系统特性
2. **频率响应**决定了滤波器对不同频率成分的处理效果，是滤波器设计的核心目标
3. **FIR滤波器**具有线性相位特性，适合需要保持信号时域特征的应用
4. **IIR滤波器**计算效率高，适合实时处理和需要陡峭过渡带的应用
5. 实际应用中，滤波器的选择需要在性能、计算复杂度和相位特性之间权衡

理解这些原理不仅有助于正确使用现有滤波器库，更重要的是能够根据具体需求设计定制化的滤波器解决方案。随着深度学习的发展，基于数据驱动的自适应滤波器正在成为新的研究热点，但经典滤波理论仍然是信号处理工程师不可或缺的工具。
