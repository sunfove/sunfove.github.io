# 激光谐振腔设计：从稳定性条件到模式分析

## 引言

激光谐振腔是激光器的核心部件，它决定了激光的输出特性，包括光束质量、功率、频率稳定性和模式结构。一个设计良好的谐振腔能够确保激光器高效、稳定地运行，而设计不当则可能导致模式竞争、功率下降甚至无法起振。从连续波激光器到超快脉冲激光器，谐振腔的设计都是决定性能的关键。本文将深入探讨谐振腔的稳定性理论、横纵模结构，并通过数学推导和Python代码演示如何分析和设计一个稳定的谐振腔。

## 核心概念

### 谐振腔稳定性

谐振腔通常由两个反射镜组成，光在两个镜面之间来回反射形成驻波。稳定性是指光在腔内经过多次往返后仍然能够被限制在镜面附近，不会从侧面逸出。一个不稳定的腔会导致光逐渐发散，无法形成稳定的激光振荡。

稳定性由腔的几何参数决定，包括腔长 $L$ 和两个反射镜的曲率半径 $R_1$、$R_2$。引入归一化参数 $g_1 = 1 - L/R_1$ 和 $g_2 = 1 - L/R_2$，稳定性条件为：

$$0 < g_1 g_2 < 1$$

当 $g_1 g_2 = 1$ 时，腔处于临界稳定状态（如共焦腔、平面腔等）；当 $g_1 g_2 < 0$ 或 $>1$ 时，腔不稳定。

### 横模与纵模

- **纵模**：决定了激光的纵向频率分布，对应于沿腔轴方向的驻波模式。纵模频率由腔长决定，相邻纵模的频率间隔称为自由光谱范围（FSR）。
- **横模**：决定了光束在横向的强度分布，用 TEM$_{mn}$ 表示（m, n 为横向模式指数）。横模的选择依赖于腔的几何形状，例如，稳定腔通常支持低阶横模（TEM$_{00}$），而不稳定腔则倾向于高阶横模。

### 自由光谱范围

自由光谱范围（FSR）是纵模之间的频率间隔，定义为：

$$\Delta \nu_{\text{FSR}} = \frac{c}{2nL}$$

其中 $c$ 是光速，$n$ 是腔内介质的折射率，$L$ 是腔长。FSR 决定了激光器能够输出的纵模数量，也影响激光的频率调谐范围。

## 数学原理

### 稳定性条件的推导

考虑一个由两个球面镜构成的谐振腔，镜面曲率半径分别为 $R_1$ 和 $R_2$，腔长为 $L$。光在腔内往返一次可以用 ABCD 矩阵描述。对于傍轴光线，往返矩阵为：

$$M = \begin{bmatrix} A & B \\ C & D \end{bmatrix}$$

一个稳定腔要求往返矩阵的迹满足：

$$-1 < \frac{A + D}{2} < 1$$

对于双镜腔，往返矩阵为：

$$M = \begin{bmatrix} 1 & 0 \\ -\frac{2}{R_2} & 1 \end{bmatrix} \begin{bmatrix} 1 & L \\ 0 & 1 \end{bmatrix} \begin{bmatrix} 1 & 0 \\ -\frac{2}{R_1} & 1 \end{bmatrix} \begin{bmatrix} 1 & L \\ 0 & 1 \end{bmatrix}$$

计算可得：

$$A = 1 - \frac{2L}{R_2}, \quad D = 1 - \frac{2L}{R_1} - \frac{2L}{R_2} + \frac{4L^2}{R_1 R_2}$$

代入稳定性条件，整理后得到：

$$0 < \left(1 - \frac{L}{R_1}\right) \left(1 - \frac{L}{R_2}\right) < 1$$

即 $0 < g_1 g_2 < 1$。

### 纵模频率

纵模频率由驻波条件决定：光在腔内往返一次，相位变化应为 $2\pi$ 的整数倍。对于长度为 $L$ 的腔，纵模频率为：

$$\nu_q = \frac{q c}{2 n L}$$

其中 $q$ 为整数（纵模序数）。相邻纵模的频率间隔即为 FSR。

## 代码实现

下面用 Python 实现一个谐振腔稳定性分析和纵模计算工具。该代码可以计算给定腔参数的稳定性，并画出稳定性图。

```python
import numpy as np
import matplotlib.pyplot as plt

def cavity_stability(R1, R2, L):
    """
    计算谐振腔的稳定性参数 g1, g2 和乘积 g1*g2
    :param R1: 镜1曲率半径 (m)
    :param R2: 镜2曲率半径 (m)
    :param L: 腔长 (m)
    :return: g1, g2, product
    """
    g1 = 1 - L / R1
    g2 = 1 - L / R2
    product = g1 * g2
    return g1, g2, product

def check_stability(product):
    """
    判断稳定性
    :param product: g1*g2
    :return: 稳定状态描述
    """
    if 0 < product < 1:
        return "稳定"
    elif product == 0 or product == 1:
        return "临界稳定"
    else:
        return "不稳定"

def mode_frequencies(L, n=1.0, q_min=1, q_max=10):
    """
    计算纵模频率
    :param L: 腔长 (m)
    :param n: 折射率
    :param q_min: 最小纵模序数
    :param q_max: 最大纵模序数
    :return: 频率数组 (Hz)
    """
    c = 3e8  # 光速 m/s
    fsr = c / (2 * n * L)
    freqs = np.array([q * fsr for q in range(q_min, q_max+1)])
    return freqs, fsr

# 示例：平凹腔 (R1=∞, R2=1m, L=0.5m)
R1 = 1e10  # 近似平面镜
R2 = 1.0
L = 0.5

g1, g2, product = cavity_stability(R1, R2, L)
status = check_stability(product)
print(f"g1 = {g1:.3f}, g2 = {g2:.3f}, g1*g2 = {product:.3f}")
print(f"腔状态: {status}")

# 计算纵模
freqs, fsr = mode_frequencies(L, n=1.0, q_min=1, q_max=5)
print(f"自由光谱范围 (FSR): {fsr/1e6:.2f} MHz")
print("纵模频率 (MHz):", freqs/1e6)

# 绘制稳定性图
R1_vals = np.linspace(0.5, 2.0, 100)
R2_vals = np.linspace(0.5, 2.0, 100)
L_fixed = 0.5
X, Y = np.meshgrid(R1_vals, R2_vals)
g1_grid = 1 - L_fixed / X
g2_grid = 1 - L_fixed / Y
product_grid = g1_grid * g2_grid

plt.figure(figsize=(8, 6))
contour = plt.contourf(X, Y, product_grid, levels=[0, 1, 2], 
                       colors=['lightgreen', 'lightcoral', 'lightblue'])
plt.colorbar(contour, label='g1*g2')
plt.xlabel('R1 (m)')
plt.ylabel('R2 (m)')
plt.title('谐振腔稳定性图 (L=0.5m)')
# 标注稳定区域 (0<g1*g2<1)
plt.text(0.7, 0.7, '稳定区域', fontsize=12, color='green', ha='center')
plt.text(1.5, 1.5, '不稳定区域', fontsize=12, color='red', ha='center')
plt.grid(True, alpha=0.3)
plt.show()
```

**代码说明**：
1. `cavity_stability` 函数计算 $g_1$ 和 $g_2$ 参数。
2. `check_stability` 根据乘积判断稳定性。
3. `mode_frequencies` 计算纵模频率和 FSR。
4. 示例以平凹腔（平面镜 + 曲率半径 1m 的凹面镜，腔长 0.5m）为例，输出稳定状态和纵模频率。
5. 最后绘制稳定性图，绿色区域表示稳定，红色表示不稳定。

## 应用场景

### 1. 固体激光器的设计

在 Nd:YAG 激光器中，常用平凹腔或凹凸腔。例如，一个典型的设计参数为：$R_1 = \infty$（平面镜），$R_2 = 1$ m，$L = 0.5$ m。计算得 $g_1=1$，$g_2=0.5$，乘积为 0.5，处于稳定区。此时 FSR 为 300 MHz，适用于单纵模或窄线宽输出。

### 2. 超快激光器中的色散管理

在飞秒激光器中，谐振腔设计需要考虑色散补偿。通常采用啁啾镜或棱镜对来管理群延迟色散。腔长 $L$ 的选择直接影响重复频率 $f_{\text{rep}} = c/(2L)$，例如，1 m 腔长对应 150 MHz 重复频率。稳定性分析确保脉冲在腔内稳定振荡，避免高阶横模导致脉冲畸变。

## 总结

本文深入探讨了激光谐振腔设计的核心原理，包括：
- **稳定性条件**：$0 < g_1 g_2 < 1$，由腔的几何参数决定。
- **纵模结构**：频率由 $q c/(2nL)$ 给出，FSR 决定了模式间隔。
- **横模影响**：稳定腔倾向于低阶横模，有利于光束质量。

通过 Python 代码，我们实现了稳定性计算和纵模分析，并展示了如何绘制稳定性图。在实际应用中，谐振腔设计需要综合考虑增益介质、输出功率、光束质量等因素。理解这些基本原理是设计高性能激光器的基础。
