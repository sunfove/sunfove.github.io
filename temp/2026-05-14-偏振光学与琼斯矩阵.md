# 偏振光学与琼斯矩阵：从光波振动到矩阵变换

## 引言

光，作为电磁波，其电场矢量在垂直于传播方向的平面内振动。这个振动方向与振幅的分布，就是我们常说的**偏振态**。从液晶显示到量子通信，从遥感探测到激光雷达，偏振态的控制与测量几乎渗透到现代光学技术的每一个角落。然而，如何用数学语言优雅地描述偏振态及其经过光学元件后的变化？答案是琼斯矩阵。本文将带领你从光的矢量表示出发，深入琼斯矢量和琼斯矩阵的数学原理，并通过 Python 代码实现一个完整的偏振模拟器。

## 核心概念

### 1. 偏振态

光波是横波，其电场矢量 $\mathbf{E}$ 在垂直于传播方向（设为 $z$ 轴）的平面内振动。在任意时刻，电场矢量可以分解为 $x$ 和 $y$ 方向的分量：

$$
\mathbf{E}(z, t) = \begin{bmatrix} E_{0x} e^{i(kz - \omega t + \phi_x)} \\ E_{0y} e^{i(kz - \omega t + \phi_y)} \end{bmatrix}
$$

其中 $E_{0x}$ 和 $E_{0y}$ 是振幅，$\phi_x$ 和 $\phi_y$ 是相位。根据这两个分量的振幅比和相位差 $\delta = \phi_y - \phi_x$，光波呈现出不同的偏振态：
- **线偏振**：$\delta = 0$ 或 $\pi$
- **圆偏振**：$E_{0x} = E_{0y}$ 且 $\delta = \pm \pi/2$
- **椭圆偏振**：一般情况

### 2. 琼斯矢量

1941年，R. C. Jones 提出用复数矢量来表示偏振态，即**琼斯矢量**。忽略时间因子 $e^{-i\omega t}$ 和空间因子 $e^{i k z}$，琼斯矢量定义为：

$$
\mathbf{J} = \begin{bmatrix} E_x \\ E_y \end{bmatrix} = \begin{bmatrix} A_x e^{i\phi_x} \\ A_y e^{i\phi_y} \end{bmatrix}
$$

例如，沿 $x$ 轴振动的线偏振光：$\mathbf{J} = \begin{bmatrix} 1 \\ 0 \end{bmatrix}$；右旋圆偏振光：$\mathbf{J} = \frac{1}{\sqrt{2}} \begin{bmatrix} 1 \\ -i \end{bmatrix}$。

### 3. 琼斯矩阵

光学元件（如偏振片、波片）对入射光偏振态的影响，可以通过一个 $2 \times 2$ 的复数矩阵来描述，即**琼斯矩阵** $\mathbf{M}$。设入射光的琼斯矢量为 $\mathbf{J}_{\text{in}}$，出射光的琼斯矢量为 $\mathbf{J}_{\text{out}}$，则有：

$$
\mathbf{J}_{\text{out}} = \mathbf{M} \cdot \mathbf{J}_{\text{in}}
$$

常见光学元件的琼斯矩阵：
- **线偏振片**（透光轴沿 $x$ 轴）：$\mathbf{M}_{\text{pol}} = \begin{bmatrix} 1 & 0 \\ 0 & 0 \end{bmatrix}$
- **四分之一波片**（快轴沿 $x$ 轴）：$\mathbf{M}_{\text{QWP}} = \begin{bmatrix} 1 & 0 \\ 0 & -i \end{bmatrix}$
- **半波片**（快轴沿 $x$ 轴）：$\mathbf{M}_{\text{HWP}} = \begin{bmatrix} 1 & 0 \\ 0 & -1 \end{bmatrix}$

## 数学原理

### 琼斯矢量的归一化与变换

为了便于比较，通常将琼斯矢量归一化，使其模长为1。例如，与 $x$ 轴成 $\theta$ 角的线偏振光：

$$
\mathbf{J}(\theta) = \begin{bmatrix} \cos\theta \\ \sin\theta \end{bmatrix}
$$

考虑一个旋转光学元件。若元件本身在自然坐标系下的琼斯矩阵为 $\mathbf{M}_0$，当它绕光轴旋转角度 $\theta$ 时，其琼斯矩阵变为：

$$
\mathbf{M}(\theta) = \mathbf{R}(-\theta) \cdot \mathbf{M}_0 \cdot \mathbf{R}(\theta)
$$

其中旋转矩阵 $\mathbf{R}(\theta)$ 定义为：

$$
\mathbf{R}(\theta) = \begin{bmatrix} \cos\theta & \sin\theta \\ -\sin\theta & \cos\theta \end{bmatrix}
$$

### 多元件级联

当光依次通过多个光学元件时，总的琼斯矩阵是各元件矩阵的乘积（按光路顺序从右向左相乘）：

$$
\mathbf{J}_{\text{out}} = \mathbf{M}_n \cdots \mathbf{M}_2 \cdot \mathbf{M}_1 \cdot \mathbf{J}_{\text{in}}
$$

## 代码实现

下面我们用 Python 实现一个完整的偏振模拟器，包括琼斯矢量定义、常见光学元件矩阵、以及模拟光经过偏振片和四分之一波片后的偏振态变化。

```python
import numpy as np
import cmath

class JonesVector:
    """琼斯矢量类"""
    def __init__(self, Ex, Ey):
        self.vector = np.array([complex(Ex), complex(Ey)])
    
    def normalize(self):
        """归一化"""
        norm = np.linalg.norm(self.vector)
        if norm > 0:
            self.vector /= norm
        return self
    
    def get_stokes(self):
        """计算斯托克斯参数（用于判断偏振态）"""
        Ex, Ey = self.vector
        S0 = abs(Ex)**2 + abs(Ey)**2
        S1 = abs(Ex)**2 - abs(Ey)**2
        S2 = 2 * np.real(Ex * np.conj(Ey))
        S3 = -2 * np.imag(Ex * np.conj(Ey))
        return S0, S1, S2, S3
    
    def __repr__(self):
        return f"JonesVector({self.vector})"

class JonesMatrix:
    """琼斯矩阵类"""
    def __init__(self, matrix):
        self.matrix = np.array(matrix, dtype=complex)
    
    def rotate(self, theta):
        """旋转矩阵 theta 弧度"""
        R = np.array([[np.cos(theta), np.sin(theta)],
                      [-np.sin(theta), np.cos(theta)]])
        self.matrix = R.T @ self.matrix @ R
        return self
    
    def apply(self, jones_vector):
        """应用于琼斯矢量"""
        new_vector = self.matrix @ jones_vector.vector
        return JonesVector(new_vector[0], new_vector[1])
    
    def __matmul__(self, other):
        """矩阵乘法（级联）"""
        if isinstance(other, JonesMatrix):
            return JonesMatrix(self.matrix @ other.matrix)
        elif isinstance(other, JonesVector):
            return self.apply(other)
        else:
            raise TypeError("Unsupported operand type")

# 定义常见光学元件
def polarizer(axis=0):
    """透光轴沿 axis 方向的线偏振片"""
    M = np.array([[1, 0], [0, 0]])
    mat = JonesMatrix(M)
    if axis != 0:
        mat.rotate(axis)
    return mat

def quarter_wave_plate(fast_axis=0):
    """快轴沿 fast_axis 方向的四分之一波片"""
    M = np.array([[1, 0], [0, -1j]])
    mat = JonesMatrix(M)
    if fast_axis != 0:
        mat.rotate(fast_axis)
    return mat

def half_wave_plate(fast_axis=0):
    """快轴沿 fast_axis 方向的半波片"""
    M = np.array([[1, 0], [0, -1]])
    mat = JonesMatrix(M)
    if fast_axis != 0:
        mat.rotate(fast_axis)
    return mat

# 示例：模拟光经过偏振片和四分之一波片
if __name__ == "__main__":
    # 入射光：45度线偏振光
    J_in = JonesVector(np.cos(np.pi/4), np.sin(np.pi/4))
    print(f"入射光: {J_in.vector}")
    
    # 经过透光轴沿x轴的偏振片
    M1 = polarizer(axis=0)
    J_mid = M1.apply(J_in)
    print(f"经过偏振片后: {J_mid.vector}")
    
    # 再经过快轴沿x轴的四分之一波片
    M2 = quarter_wave_plate(fast_axis=0)
    J_out = M2.apply(J_mid)
    print(f"经过四分之一波片后: {J_out.vector}")
    
    # 级联方式
    M_total = M2 @ M1
    J_out2 = M_total.apply(J_in)
    print(f"级联计算验证: {J_out2.vector}")
    
    # 检查偏振态
    S0, S1, S2, S3 = J_out.get_stokes()
    print(f"斯托克斯参数: S0={S0:.3f}, S1={S1:.3f}, S2={S2:.3f}, S3={S3:.3f}")
    if abs(S3/S0) > 0.99:
        print("输出为圆偏振光")
    elif abs(S1/S0) > 0.99:
        print("输出为线偏振光")
    else:
        print("输出为椭圆偏振光")
```

## 应用场景

### 1. 液晶显示（LCD）中的偏振控制

液晶显示器的核心原理就是利用液晶分子对偏振光的旋转效应。典型的 LCD 结构包含：
- 背光源发出自然光
- 第一片偏振片将其变为线偏振光
- 液晶层在电场作用下旋转偏振面
- 第二片偏振片（检偏器）控制透光量

通过琼斯矩阵，我们可以精确计算每个像素的亮度。例如，当液晶层旋转偏振面 90° 时，光能够通过正交的检偏器；当液晶层不旋转时，光被完全阻挡。

### 2. 偏振态测量与椭偏仪

在材料科学中，椭偏仪通过测量反射光的偏振态变化来推断薄膜的厚度和折射率。其核心是测量入射光和反射光的琼斯矢量变化。假设入射光为已知偏振态 $\mathbf{J}_{\text{in}}$，反射后变为 $\mathbf{J}_{\text{out}}$，则反射过程的琼斯矩阵 $\mathbf{M}_{\text{ref}}$ 包含了样品的光学信息：

$$
\mathbf{J}_{\text{out}} = \mathbf{M}_{\text{ref}} \cdot \mathbf{J}_{\text{in}}
$$

通过旋转波片和偏振片，测量多个 $\mathbf{J}_{\text{out}}$，可以反演出 $\mathbf{M}_{\text{ref}}$ 的四个元素，从而得到样品的椭偏参数 $\Psi$ 和 $\Delta$。

## 总结

本文从电磁波的基本振动出发，引入了琼斯矢量和琼斯矩阵这两个强大的数学工具。关键要点回顾：

1. **琼斯矢量**：用复数向量 $\begin{bmatrix} E_x \\ E_y \end{bmatrix}$ 简洁地表示任意偏振态
2. **琼斯矩阵**：$2 \times 2$ 复数矩阵描述光学元件对偏振态的线性变换
3. **级联原理**：多个元件的总矩阵按光路顺序从右向左相乘
4. **旋转变换**：元件旋转时，通过 $\mathbf{R}(-\theta) \mathbf{M}_0 \mathbf{R}(\theta)$ 得到旋转后的矩阵

琼斯矩阵方法虽然不能处理部分偏振光和退偏振效应（这时需要改用穆勒矩阵），但在处理完全偏振光时，它提供了最简洁、最直观的数学框架。无论是理解液晶显示原理，还是设计量子光学实验，掌握琼斯矩阵都是必不可少的一步。
