# 贝叶斯优化：当黑盒函数遇上智能搜索

## 1. 引言

在机器学习与工程优化中，我们经常面临这样的困境：需要优化的目标函数是一个“黑盒”——我们无法获得它的梯度，每次评估成本极高（例如训练一次深度学习模型可能需要数小时），而且函数本身可能带有噪声。传统的网格搜索或随机搜索在这些场景下效率低下，而梯度下降法又无从施展。贝叶斯优化（Bayesian Optimization）正是为解决这类问题而生。它通过构建一个概率代理模型来模拟未知函数，并利用采集函数智能地选择下一个评估点，从而在尽可能少的评估次数内找到全局最优解。本文将深入解析贝叶斯优化的核心原理、数学推导，并提供一个完整的Python实现。

## 2. 核心概念

### 2.1 代理模型 (Surrogate Model)
代理模型是贝叶斯优化的“大脑”。由于我们无法直接了解目标函数 $$f(x)$$ 的全貌，我们用一个概率模型来近似它。最常用的代理模型是**高斯过程 (Gaussian Process, GP)**。GP 假设任意有限个点的函数值服从联合高斯分布，因此它不仅能给出任意点 $$x$$ 的预测均值 $$\mu(x)$$，还能给出预测的不确定性（方差）$$\sigma^2(x)$$。这种“知道哪里不确定”的能力是贝叶斯优化的关键。

### 2.2 采集函数 (Acquisition Function)
采集函数是贝叶斯优化的“策略制定者”。它基于当前代理模型的预测，决定下一个采样点应该在哪里。采集函数需要在“利用”（在当前最优区域附近搜索）和“探索”（在不确定性高的区域搜索）之间取得平衡。常见的采集函数包括：
- **期望改进 (Expected Improvement, EI)**：衡量某个点相对于当前最优值的期望提升。
- **置信上界 (Upper Confidence Bound, UCB)**：在均值基础上加上一个与方差相关的奖励项。

### 2.3 高斯过程 (Gaussian Process)
高斯过程是定义在连续函数空间上的随机过程。一个 GP 完全由均值函数 $$m(x)$$ 和协方差函数（核函数）$$k(x, x')$$ 确定：
$$f(x) \sim \mathcal{GP}(m(x), k(x, x'))$$
通常我们设 $$m(x)=0$$，而核函数的选择决定了函数的平滑性。最常用的核是平方指数核（RBF）：
$$k(x_i, x_j) = \sigma_f^2 \exp\left(-\frac{||x_i - x_j||^2}{2l^2}\right)$$
其中 $$\sigma_f^2$$ 是信号方差，$$l$$ 是长度尺度参数。

## 3. 数学原理

### 3.1 高斯过程预测
假设我们有观测数据 $$\mathcal{D} = \{(x_i, y_i)\}_{i=1}^n$$，其中 $$y_i = f(x_i) + \epsilon$$，$$\epsilon \sim \mathcal{N}(0, \sigma_n^2)$$。对于一个新的输入点 $$x_*$$，GP 的后验预测分布为：
$$\begin{aligned}
\mu(x_*) &= \mathbf{k}_*^\top (K + \sigma_n^2 I)^{-1} \mathbf{y} \\
\sigma^2(x_*) &= k(x_*, x_*) - \mathbf{k}_*^\top (K + \sigma_n^2 I)^{-1} \mathbf{k}_*
\end{aligned}$$
其中：
- $$K$$ 是 $$n \times n$$ 的核矩阵，$$K_{ij} = k(x_i, x_j)$$
- $$\mathbf{k}_*$$ 是 $$n \times 1$$ 的向量，$$(\mathbf{k}_*)_i = k(x_*, x_i)$$
- $$\mathbf{y} = [y_1, ..., y_n]^\top$$

### 3.2 期望改进 (EI)
设当前最优观测值为 $$f^* = \max(y_1, ..., y_n)$$。对于任意点 $$x$$，其改进量定义为：
$$I(x) = \max(0, f(x) - f^*)$$
由于 $$f(x) \sim \mathcal{N}(\mu(x), \sigma^2(x))$$，EI 的解析形式为：
$$\begin{aligned}
\text{EI}(x) &= \mathbb{E}[I(x)] \\
&= \begin{cases}
(\mu(x) - f^*) \Phi\left(\frac{\mu(x) - f^*}{\sigma(x)}\right) + \sigma(x) \phi\left(\frac{\mu(x) - f^*}{\sigma(x)}\right) & \text{if } \sigma(x) > 0 \\
0 & \text{if } \sigma(x) = 0
\end{cases}
\end{aligned}$$
其中 $$\Phi$$ 和 $$\phi$$ 分别是标准正态分布的 CDF 和 PDF。EI 自然地平衡了利用（当 $$\mu(x) > f^*$$ 时，第一项主导）和探索（当 $$\sigma(x)$$ 大时，第二项主导）。

### 3.3 优化流程
贝叶斯优化的迭代过程如下：
1. 初始化：用少量随机点评估目标函数，构建初始 GP 模型。
2. 循环直到预算耗尽：
   - 最大化采集函数 $$a(x)$$ 找到下一个候选点 $$x_{next} = \arg\max_x a(x)$$
   - 评估目标函数 $$y_{next} = f(x_{next})$$
   - 更新 GP 模型，加入新观测数据

## 4. 代码实现

下面是一个完整的贝叶斯优化实现，用于优化一个一维测试函数。

```python
import numpy as np
from scipy.stats import norm
from scipy.optimize import minimize
import matplotlib.pyplot as plt

# 定义目标函数（带噪声）
def objective_function(x, noise=0.1):
    # 这是一个多峰函数，全局最优在 x≈0.5 附近
    return np.sin(5 * x) * (1 - np.tanh(x**2)) + np.random.normal(0, noise)

# 定义RBF核函数
def rbf_kernel(x1, x2, l=0.1, sigma_f=1.0):
    sq_dist = np.sum((x1 - x2)**2)
    return sigma_f**2 * np.exp(-0.5 * sq_dist / l**2)

# 高斯过程预测
def gp_predict(X_train, y_train, X_test, l=0.1, sigma_f=1.0, sigma_n=0.1):
    n = len(X_train)
    # 构建核矩阵
    K = np.zeros((n, n))
    for i in range(n):
        for j in range(n):
            K[i, j] = rbf_kernel(X_train[i], X_train[j], l, sigma_f)
    K += sigma_n**2 * np.eye(n)  # 添加噪声项
    
    # 计算逆矩阵（实际应用中使用Cholesky分解更稳定）
    K_inv = np.linalg.inv(K)
    
    # 预测
    mu = np.zeros(len(X_test))
    var = np.zeros(len(X_test))
    for i, x in enumerate(X_test):
        k_star = np.array([rbf_kernel(x, x_train, l, sigma_f) for x_train in X_train])
        mu[i] = k_star @ K_inv @ y_train
        var[i] = rbf_kernel(x, x, l, sigma_f) - k_star @ K_inv @ k_star
    
    return mu, var

# 期望改进采集函数
def expected_improvement(X_train, y_train, X_test, xi=0.01):
    mu, var = gp_predict(X_train, y_train, X_test)
    sigma = np.sqrt(np.maximum(var, 1e-10))  # 防止数值问题
    f_best = np.max(y_train)
    
    # 计算改进
    improvement = mu - f_best - xi
    z = improvement / sigma
    
    ei = improvement * norm.cdf(z) + sigma * norm.pdf(z)
    ei[sigma == 0] = 0
    return ei

# 贝叶斯优化主循环
def bayesian_optimization(n_iterations=15, n_init=3):
    # 初始化随机点
    X_train = np.random.uniform(0, 1, n_init)
    y_train = np.array([objective_function(x) for x in X_train])
    
    # 记录最优值
    best_y = np.max(y_train)
    best_x = X_train[np.argmax(y_train)]
    
    for i in range(n_iterations):
        # 在密集网格上优化采集函数
        X_candidates = np.linspace(0, 1, 1000)
        ei_values = expected_improvement(X_train, y_train, X_candidates)
        
        # 选择EI最大的点
        x_next = X_candidates[np.argmax(ei_values)]
        
        # 评估目标函数
        y_next = objective_function(x_next)
        
        # 更新训练集
        X_train = np.append(X_train, x_next)
        y_train = np.append(y_train, y_next)
        
        # 更新最优值
        if y_next > best_y:
            best_y = y_next
            best_x = x_next
        
        print(f"Iteration {i+1}: Best f(x) = {best_y:.4f} at x = {best_x:.4f}")
    
    return best_x, best_y, X_train, y_train

# 运行优化
best_x, best_y, X_train, y_train = bayesian_optimization(n_iterations=20)

# 可视化结果
X_plot = np.linspace(0, 1, 100)
mu, var = gp_predict(X_train, y_train, X_plot)
std = np.sqrt(var)

plt.figure(figsize=(12, 5))
plt.subplot(1, 2, 1)
plt.plot(X_plot, objective_function(X_plot, noise=0), 'k--', label='True function')
plt.scatter(X_train, y_train, c='r', label='Evaluated points')
plt.plot(X_plot, mu, 'b-', label='GP mean')
plt.fill_between(X_plot, mu - 2*std, mu + 2*std, alpha=0.2, label='95% CI')
plt.xlabel('x')
plt.ylabel('f(x)')
plt.legend()
plt.title('Bayesian Optimization Progress')

plt.subplot(1, 2, 2)
ei = expected_improvement(X_train, y_train, X_plot)
plt.plot(X_plot, ei, 'g-', label='Expected Improvement')
plt.xlabel('x')
plt.ylabel('EI')
plt.legend()
plt.title('Acquisition Function')
plt.tight_layout()
plt.show()
```

## 5. 应用场景

### 5.1 超参数调优
这是贝叶斯优化最经典的应用。假设我们要调整一个随机森林模型的 `n_estimators` 和 `max_depth` 两个超参数。传统的网格搜索需要尝试所有组合，而贝叶斯优化通过构建 GP 模型，通常在 20-30 次评估内就能找到接近最优的参数组合，比网格搜索效率高 10 倍以上。

### 5.2 机器人控制参数优化
在机器人运动控制中，PID 控制器的三个参数（比例、积分、微分增益）需要精细调整。每次评估需要让机器人在真实环境中运行一段轨迹，成本极高。贝叶斯优化可以在 50 次左右的实验中找到最优参数，而人工调参可能需要数百次。

## 6. 总结

贝叶斯优化是一种强大的黑盒优化方法，其核心优势在于：
1. **样本效率高**：通过概率模型智能选择采样点，大幅减少评估次数
2. **处理噪声**：GP 模型天然支持带噪观测
3. **全局搜索**：采集函数自动平衡探索与利用

关键要点：
- 代理模型（GP）提供了预测均值和不确定性
- 采集函数（如 EI）指导采样策略
- 核函数的选择对性能有重要影响
- 实际应用中需注意超参数（如核函数长度尺度）的调优

贝叶斯优化并非万能——它在高维问题（维度 > 20）和高度多峰函数上会遇到困难。但对于中等维度的昂贵黑盒优化问题，它仍然是目前最先进的解决方案之一。
