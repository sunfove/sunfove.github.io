# 小波分析与应用：从傅里叶到多尺度时频分析

## 引言

在信号处理领域，傅里叶变换一直是分析信号频率特性的基石。然而，傅里叶变换存在一个根本性缺陷：它假设信号是平稳的，无法同时提供时间和频率的局部化信息。对于非平稳信号（如地震波、心电信号、语音信号），这一局限性尤为明显。小波分析(Wavelet Analysis)应运而生，它通过可伸缩、可平移的基函数，在时域和频域同时实现多分辨率分析，成为处理非平稳信号的强大工具。本文将深入探讨小波变换的核心概念、数学原理、代码实现及应用场景。

## 核心概念

### 1. 连续小波变换 (CWT)

连续小波变换通过将信号与一个可伸缩、可平移的小波基函数进行内积，获得信号在时间-尺度平面上的表示。其核心思想是：使用一个母小波 $\psi(t)$，通过缩放参数 $a$ 和平移参数 $b$ 生成一系列子小波：

$$\psi_{a,b}(t) = \frac{1}{\sqrt{|a|}} \psi\left(\frac{t-b}{a}\right)$$

其中 $a$ 控制小波的频率（尺度），$b$ 控制小波在时间轴上的位置。CWT 提供了冗余的、连续的时间-频率表示，便于理论分析。

### 2. 离散小波变换 (DWT)

DWT 通过离散化尺度参数 $a$ 和平移参数 $b$（通常取 $a=2^j, b=k2^j$，其中 $j,k \in \mathbb{Z}$），形成正交小波基，实现信号的高效分解和重构。DWT 的核心思想是 Mallat 算法：通过低通和高通滤波器级联，将信号分解为近似系数（低频）和细节系数（高频），实现多分辨率分解。

### 3. 多分辨率分析 (MRA)

MRA 是小波分析的理论框架，由 Mallat 和 Meyer 提出。它通过一系列嵌套的闭子空间 $\{V_j\}_{j\in\mathbb{Z}}$ 来逼近 $L^2(\mathbb{R})$ 空间，其中每个子空间 $V_j$ 对应尺度 $2^j$ 下的信号近似。MRA 的核心是尺度函数 $\phi(t)$ 和小波函数 $\psi(t)$ 满足双尺度方程：

$$\phi(t) = \sum_n h[n] \phi(2t - n)$$
$$\psi(t) = \sum_n g[n] \phi(2t - n)$$

其中 $h[n]$ 和 $g[n]$ 分别是低通和高通滤波器系数，且 $g[n] = (-1)^n h[1-n]$。

## 数学原理

### 连续小波变换定义

对于信号 $x(t) \in L^2(\mathbb{R})$，其连续小波变换定义为：

$$W_x(a,b) = \langle x, \psi_{a,b} \rangle = \frac{1}{\sqrt{|a|}} \int_{-\infty}^{\infty} x(t) \psi^*\left(\frac{t-b}{a}\right) dt$$

其中 $a \neq 0$ 是尺度参数，$b$ 是平移参数，$\psi^*(t)$ 是母小波的复共轭。逆变换公式为：

$$x(t) = \frac{1}{C_\psi} \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} \frac{W_x(a,b)}{a^2} \psi_{a,b}(t) \, da \, db$$

其中 $C_\psi = \int_{-\infty}^{\infty} \frac{|\hat{\psi}(\omega)|^2}{|\omega|} d\omega < \infty$ 是容许性条件。

### 离散小波变换分解

基于 MRA 理论，DWT 的分解过程通过 Mallat 算法实现。对于离散信号 $x[n]$，其第 $j$ 层的分解为：

**近似系数**：
$$cA_j[k] = \sum_n cA_{j-1}[n] \cdot h[2k - n]$$

**细节系数**：
$$cD_j[k] = \sum_n cA_{j-1}[n] \cdot g[2k - n]$$

其中 $cA_0[n] = x[n]$，$h[n]$ 是低通滤波器，$g[n]$ 是高通滤波器。重构过程为：

$$cA_{j-1}[k] = \sum_n cA_j[n] \cdot \tilde{h}[k - 2n] + \sum_n cD_j[n] \cdot \tilde{g}[k - 2n]$$

其中 $\tilde{h}[n]$ 和 $\tilde{g}[n]$ 是重构滤波器。

## 代码实现

以下 Python 代码使用 PyWavelets 库实现 DWT 分解、阈值去噪和重构：

```python
import numpy as np
import matplotlib.pyplot as plt
import pywt

def generate_noisy_signal(t):
    """生成含噪声的非平稳信号"""
    # 基频信号
    clean = np.sin(2 * np.pi * 5 * t) + np.sin(2 * np.pi * 20 * t)
    # 添加高斯白噪声
    noise = 0.5 * np.random.randn(len(t))
    noisy = clean + noise
    return clean, noisy

def wavelet_denoise(signal, wavelet='db4', level=3, threshold_mode='soft'):
    """
    小波阈值去噪
    Args:
        signal: 含噪信号
        wavelet: 小波基名称
        level: 分解层数
        threshold_mode: 'soft' 或 'hard'
    Returns:
        denoised: 去噪后信号
    """
    # 1. DWT分解
    coeffs = pywt.wavedec(signal, wavelet, level=level)
    
    # 2. 估计阈值（使用通用阈值公式）
    # 阈值 = sigma * sqrt(2 * log(N))，其中 sigma 是噪声标准差
    sigma = np.median(np.abs(coeffs[-1])) / 0.6745  # 鲁棒估计
    threshold = sigma * np.sqrt(2 * np.log(len(signal)))
    
    # 3. 对细节系数进行阈值处理（保留近似系数）
    coeffs_thresh = [coeffs[0]]  # 保留近似系数
    for i in range(1, len(coeffs)):
        if threshold_mode == 'soft':
            # 软阈值：收缩系数
            coeffs_thresh.append(pywt.threshold(coeffs[i], threshold, 'soft'))
        else:
            # 硬阈值：保留或置零
            coeffs_thresh.append(pywt.threshold(coeffs[i], threshold, 'hard'))
    
    # 4. 重构信号
    denoised = pywt.waverec(coeffs_thresh, wavelet)
    return denoised

# 生成测试信号
fs = 1000  # 采样频率
t = np.arange(0, 2, 1/fs)  # 2秒时长
clean, noisy = generate_noisy_signal(t)

# 应用小波去噪
denoised = wavelet_denoise(noisy, wavelet='db8', level=4, threshold_mode='soft')

# 计算信噪比提升
def snr(original, processed):
    noise = original - processed
    return 10 * np.log10(np.sum(original**2) / np.sum(noise**2))

snr_before = snr(clean, noisy)
snr_after = snr(clean, denoised)

print(f"去噪前 SNR: {snr_before:.2f} dB")
print(f"去噪后 SNR: {snr_after:.2f} dB")

# 可视化结果
plt.figure(figsize=(12, 8))
plt.subplot(3, 1, 1)
plt.plot(t, clean, 'b', label='原始信号')
plt.legend()
plt.title('干净信号')

plt.subplot(3, 1, 2)
plt.plot(t, noisy, 'r', label='含噪信号', alpha=0.7)
plt.plot(t, clean, 'b--', label='原始信号', alpha=0.5)
plt.legend()
plt.title(f'含噪信号 (SNR={snr_before:.2f} dB)')

plt.subplot(3, 1, 3)
plt.plot(t, denoised, 'g', label='去噪信号')
plt.plot(t, clean, 'b--', label='原始信号', alpha=0.5)
plt.legend()
plt.title(f'小波去噪结果 (SNR={snr_after:.2f} dB)')

plt.tight_layout()
plt.show()
```

## 应用场景

### 1. 地震信号处理

地震信号是典型的非平稳信号，包含P波、S波和面波等不同频率成分。小波分析在地震学中有以下应用：

- **震相识别**：利用 DWT 的多分辨率特性，可以自动识别 P 波和 S 波的到达时间。P 波通常表现为高频成分的突然增强，而 S 波则对应低频成分的变化。
  
- **噪声压制**：地震记录中常包含环境噪声和仪器噪声，通过小波阈值去噪可以显著提高信噪比，保留重要的震相信息。

- **时频分析**：使用 CWT 生成地震信号的时频谱，可以直观地观察不同震相的能量随时间和频率的变化。

### 2. 心电图 (ECG) 信号分析

心电信号是生物医学中重要的非平稳信号，小波分析在 ECG 处理中发挥关键作用：

- **QRS 波检测**：QRS 复合波是 ECG 中最显著的特征，其频率范围约为 5-15 Hz。通过 DWT 分解，可以在特定尺度上增强 QRS 波，实现准确的心跳检测。

- **心律失常检测**：不同类型的心律失常（如房颤、室性早搏）在时频域表现出独特的模式，小波分析可以帮助自动分类这些异常模式。

- **基线漂移去除**：ECG 信号常受呼吸和运动伪影影响，产生低频基线漂移。利用 MRA 特性，可以去除低频近似分量，保留有效的心电波形。

## 总结

小波分析作为时频分析的重要工具，克服了傅里叶变换的局限性，为处理非平稳信号提供了强有力的数学框架。本文的核心要点回顾：

1. **连续小波变换 (CWT)** 提供了冗余的连续时频表示，适合理论分析和模式识别
2. **离散小波变换 (DWT)** 通过 Mallat 算法实现高效的多分辨率分解和重构
3. **多分辨率分析 (MRA)** 建立了小波理论的数学基础，统一了尺度函数和小波函数的关系
4. 小波阈值去噪是实际应用中最重要的技术之一，通过选择合适的小波基和阈值策略，可以显著提升信号质量
5. 在地震信号处理和心电信号分析等领域，小波分析展现了不可替代的价值

随着深度学习的发展，小波变换与神经网络的结合（如小波神经网络、小波变换层）正在开辟新的研究方向，值得持续关注和探索。
