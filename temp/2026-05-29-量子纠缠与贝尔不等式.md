# 量子纠缠与贝尔不等式：当爱因斯坦输给概率

## 引言

“上帝不掷骰子”——爱因斯坦这句名言背后，隐藏着他对量子力学中“超距作用”的深深不安。1935年，爱因斯坦与波多尔斯基、罗森共同提出EPR佯谬，质疑量子力学是否完备。他们认为，如果量子理论正确，那么两个纠缠粒子之间必然存在某种“幽灵般的超距作用”。直到1964年，物理学家约翰·贝尔提出了一个可实验验证的不等式——贝尔不等式，才为这场持续近30年的争论画上了句号。本文将深入探讨量子纠缠的本质，推导贝尔不等式的数学形式，并通过代码模拟其违反过程，揭示量子世界的反直觉真相。

## 核心概念

### 1. 量子纠缠

量子纠缠是指两个或多个量子系统之间的一种非经典关联，使得对其中一个系统的测量会瞬间影响另一个系统的状态，无论它们相距多远。数学上，一个两粒子系统的纠缠态可以表示为：

$$|\Psi\rangle = \frac{1}{\sqrt{2}}(|0\rangle_A|1\rangle_B - |1\rangle_A|0\rangle_B)$$

这个态是**不可分离的**，即不能写成两个子系统态的直积形式 $|\psi\rangle_A \otimes |\phi\rangle_B$。当测量粒子A得到 $|0\rangle$ 时，粒子B必然塌缩到 $|1\rangle$，反之亦然。

### 2. 局域实在论

局域实在论包含两个核心假设：
- **实在性**：物理系统的属性是客观存在的，与测量无关。
- **局域性**：一个事件不能以超光速影响另一个事件。

EPR佯谬正是基于这两个假设，认为量子力学的不完备性导致了“超距作用”的表象。

### 3. 贝尔态（最大纠缠态）

贝尔态是两粒子系统的最大纠缠态，共有四个：

$$|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$
$$|\Phi^-\rangle = \frac{1}{\sqrt{2}}(|00\rangle - |11\rangle)$$
$$|\Psi^+\rangle = \frac{1}{\sqrt{2}}(|01\rangle + |10\rangle)$$
$$|\Psi^-\rangle = \frac{1}{\sqrt{2}}(|01\rangle - |10\rangle)$$

其中 $|\Psi^-\rangle$ 是自旋单态，也是我们后面实验用的态。

## 数学原理：CHSH不等式

贝尔不等式有多种形式，其中最常用的是CHSH（Clauser-Horne-Shimony-Holt）不等式。考虑两个观测者Alice和Bob，各自可以选择两种测量方向：Alice选择 $a$ 或 $a'$，Bob选择 $b$ 或 $b'$。每次测量结果取值为 $\pm 1$。

定义关联函数：
$$E(a,b) = \langle A(a)B(b) \rangle$$

其中 $A(a)$ 是Alice在方向 $a$ 的测量结果。在局域实在论下，我们可以证明：

$$S = |E(a,b) + E(a,b') + E(a',b) - E(a',b')| \leq 2$$

这个不等式就是CHSH不等式。其推导基于局域隐变量理论：假设存在隐变量 $\lambda$，使得测量结果可以预先确定，且 $\lambda$ 的概率分布 $p(\lambda)$ 不依赖于测量设置。则：

$$E(a,b) = \int p(\lambda) A(a,\lambda) B(b,\lambda) d\lambda$$

其中 $A(a,\lambda), B(b,\lambda) \in \{\pm 1\}$。通过代数运算可得上述不等式。

然而，量子力学预测对于某些纠缠态和测量方向，$S$ 可以超过2，最大可达 $2\sqrt{2} \approx 2.828$。例如，选择测量方向：
- $a = 0^\circ, a' = 45^\circ$
- $b = 22.5^\circ, b' = 67.5^\circ$

对于 $|\Psi^-\rangle$ 态，量子力学给出：
$$E(a,b) = -\cos(\theta_a - \theta_b)$$

代入计算得 $S = 2\sqrt{2}$，明显违反CHSH不等式。

## 代码实现

下面用Python模拟CHSH实验，验证量子力学对贝尔不等式的违反。

```python
import numpy as np
import random

# 定义泡利矩阵
def pauli_matrix(theta):
    """返回沿角度theta方向的测量算符"""
    return np.array([
        [np.cos(theta), np.sin(theta)],
        [np.sin(theta), -np.cos(theta)]
    ])

# 定义贝尔态 |Psi-> = (|01> - |10>)/sqrt(2)
psi_minus = np.array([0, 1, -1, 0]) / np.sqrt(2)

def measure_entangled_state(psi, theta_a, theta_b, num_trials=10000):
    """
    对纠缠态进行测量模拟
    返回Alice和Bob的测量结果列表
    """
    results_a = []
    results_b = []
    
    for _ in range(num_trials):
        # 计算联合概率分布
        # 测量算符的投影
        M_a = pauli_matrix(theta_a)
        M_b = pauli_matrix(theta_b)
        
        # 构造联合测量算符（张量积）
        M_joint = np.kron(M_a, M_b)
        
        # 计算期望值
        expectation = np.dot(psi.conj().T, np.dot(M_joint, psi))
        
        # 根据概率分布随机生成结果
        # 对于自旋1/2系统，测量结果为+1或-1
        p_plus_plus = (1 + expectation) / 4  # 简化模型
        p_plus_minus = (1 - expectation) / 4
        p_minus_plus = (1 - expectation) / 4
        p_minus_minus = (1 + expectation) / 4
        
        r = random.random()
        if r < p_plus_plus:
            results_a.append(1)
            results_b.append(1)
        elif r < p_plus_plus + p_plus_minus:
            results_a.append(1)
            results_b.append(-1)
        elif r < p_plus_plus + p_plus_minus + p_minus_plus:
            results_a.append(-1)
            results_b.append(1)
        else:
            results_a.append(-1)
            results_b.append(-1)
    
    return results_a, results_b

def calculate_E(results_a, results_b):
    """计算关联函数 E(a,b) = <A B>"""
    return np.mean(np.array(results_a) * np.array(results_b))

def chsh_experiment():
    """执行完整的CHSH实验"""
    print("开始CHSH实验模拟...")
    
    # 测量角度（度）
    angles = {
        'a': 0,
        'a_prime': 45,
        'b': 22.5,
        'b_prime': 67.5
    }
    
    # 转换为弧度
    theta_a = np.radians(angles['a'])
    theta_a_prime = np.radians(angles['a_prime'])
    theta_b = np.radians(angles['b'])
    theta_b_prime = np.radians(angles['b_prime'])
    
    # 测量所有组合
    num_trials = 10000
    
    print(f"每个测量组合进行 {num_trials} 次实验...")
    
    # E(a,b)
    ra, rb = measure_entangled_state(psi_minus, theta_a, theta_b, num_trials)
    E_ab = calculate_E(ra, rb)
    
    # E(a,b')
    ra, rb = measure_entangled_state(psi_minus, theta_a, theta_b_prime, num_trials)
    E_ab_prime = calculate_E(ra, rb)
    
    # E(a',b)
    ra, rb = measure_entangled_state(psi_minus, theta_a_prime, theta_b, num_trials)
    E_a_prime_b = calculate_E(ra, rb)
    
    # E(a',b')
    ra, rb = measure_entangled_state(psi_minus, theta_a_prime, theta_b_prime, num_trials)
    E_a_prime_b_prime = calculate_E(ra, rb)
    
    # 计算S
    S = abs(E_ab + E_ab_prime + E_a_prime_b - E_a_prime_b_prime)
    
    print(f"\n实验结果：")
    print(f"E(a,b) = {E_ab:.4f}")
    print(f"E(a,b') = {E_ab_prime:.4f}")
    print(f"E(a',b) = {E_a_prime_b:.4f}")
    print(f"E(a',b') = {E_a_prime_b_prime:.4f}")
    print(f"S = {S:.4f}")
    print(f"量子力学预测 S = {2*np.sqrt(2):.4f}")
    print(f"局域实在论界限 S <= 2")
    
    if S > 2:
        print("\n结论：贝尔不等式被违反！量子力学胜利！")
    else:
        print("\n结论：贝尔不等式成立。")

if __name__ == "__main__":
    chsh_experiment()
```

## 应用场景

### 1. 量子密钥分发（QKD）

BB84协议利用量子纠缠实现绝对安全的通信。Alice和Bob共享纠缠粒子对，通过测量结果的相关性生成密钥。任何窃听行为都会破坏纠缠态，从而被检测到。贝尔不等式的违反确保了通信的安全性——如果Eve试图拦截，她会引入退相干，导致S值回到经典界限内。

### 2. 量子隐形传态

利用纠缠态可以实现量子信息的远程传输。1997年，Anton Zeilinger团队首次实现了量子隐形传态实验。过程如下：
1. Alice拥有一个未知量子态 $|\psi\rangle$ 和一个纠缠粒子对的一半
2. Alice对两个粒子进行贝尔态测量
3. 测量结果通过经典信道发送给Bob
4. Bob根据接收到的信息对自己的粒子进行幺正变换，恢复出原始态 $|\psi\rangle$

这个过程中，纠缠态起到了“量子通道”的作用，使得信息可以瞬间传递，但经典信道的存在保证了不违反相对论。
