# 光学薄膜设计与分析：从干涉原理到传输矩阵法

## 引言

光学薄膜是一种厚度在光波长量级的纳米级多层结构，广泛应用于抗反射涂层、高反射镜、滤光片等光学元件中。其核心原理是利用薄膜上下表面的反射光干涉，通过精确控制膜层厚度和折射率来调控光的反射、透射和吸收特性。从相机镜头的增透膜到激光谐振腔的反射镜，光学薄膜已经渗透到现代光学系统的方方面面。本文将深入探讨薄膜干涉的物理本质、传输矩阵法的数学推导，并通过Python代码实现多层膜的光学特性计算。

## 核心概念

### 薄膜干涉

当一束光入射到薄膜表面时，会在薄膜的上表面和下表面分别发生反射和折射。这两束反射光之间由于光程差而产生相位差，从而发生干涉。对于单层膜，反射率由膜层厚度d、折射率n以及入射角度θ共同决定。

### 传输矩阵法

传输矩阵法（Transfer Matrix Method, TMM）是一种处理多层膜光学特性的高效数值方法。它将每一层膜视为一个2×2的传输矩阵，通过矩阵乘法将整个多层系统的光学响应串联起来。这种方法不仅适用于正入射，还能处理任意入射角度和偏振态。

### 等效折射率

对于多层膜结构，可以将其等效为具有某种有效折射率的单层膜。等效折射率的概念在设计和优化薄膜系统时非常有用，例如在减反射膜设计中，通过调整多层膜的等效折射率来匹配空气和基底的折射率。

## 数学原理

### 单层膜反射率公式

考虑一束波长为λ的光以角度θ₀入射到折射率为n₁的薄膜上，薄膜厚度为d，基底折射率为n₂。对于TE偏振（s偏振），单层膜反射率R为：

$$
R = \frac{r_{01}^2 + r_{12}^2 + 2r_{01}r_{12}\cos(2\delta)}{1 + r_{01}^2 r_{12}^2 + 2r_{01}r_{12}\cos(2\delta)}
$$

其中，r₀₁和r₁₂分别是空气-薄膜界面和薄膜-基底界面的菲涅耳反射系数，δ是薄膜内的相位厚度：

$$
\delta = \frac{2\pi}{\lambda} n_1 d \cos\theta_1
$$

对于TM偏振（p偏振），反射系数需使用对应的菲涅耳公式。

### 传输矩阵方程

对于多层膜系统，每一层膜的传输矩阵为：

$$
M_j = \begin{bmatrix} \cos\delta_j & \frac{i}{p_j}\sin\delta_j \\ i p_j \sin\delta_j & \cos\delta_j \end{bmatrix}
$$

其中，对于TE偏振，p_j = n_j \cos\theta_j；对于TM偏振，p_j = \cos\theta_j / n_j。δ_j = (2π/λ) n_j d_j \cos\theta_j。

整个多层系统的总传输矩阵为各层矩阵的乘积：

$$
M = M_1 M_2 \cdots M_N = \begin{bmatrix} m_{11} & m_{12} \\ m_{21} & m_{22} \end{bmatrix}
$$

系统的反射系数和透射系数为：

$$
r = \frac{p_0 m_{11} + p_0 p_{N+1} m_{12} - m_{21} - p_{N+1} m_{22}}{p_0 m_{11} + p_0 p_{N+1} m_{12} + m_{21} + p_{N+1} m_{22}}
$$

$$
t = \frac{2p_0}{p_0 m_{11} + p_0 p_{N+1} m_{12} + m_{21} + p_{N+1} m_{22}}
$$

反射率和透射率分别为R = |r|²，T = |t|²。

## 代码实现

以下是用Python实现的多层膜传输矩阵法计算代码：

```python
import numpy as np
import matplotlib.pyplot as plt

def transfer_matrix(n_layer, d_layer, theta0, lam, polarization='TE'):
    """
    计算多层膜系统的传输矩阵
    
    参数:
        n_layer: 各层折射率列表 (包含入射介质和基底)
        d_layer: 各层厚度列表 (不含入射介质和基底)
        theta0: 入射角 (弧度)
        lam: 波长 (nm)
        polarization: 'TE' 或 'TM'
    
    返回:
        M: 总传输矩阵 (2x2)
    """
    # 使用斯涅尔定律计算各层折射角
    theta = [theta0]
    for i in range(1, len(n_layer)):
        theta.append(np.arcsin(n_layer[i-1] * np.sin(theta[i-1]) / n_layer[i]))
    
    # 初始化单位矩阵
    M = np.eye(2, dtype=complex)
    
    # 遍历各层 (不包括入射介质和基底)
    for j in range(len(d_layer)):
        # 计算相位厚度
        delta = 2 * np.pi * n_layer[j+1] * d_layer[j] * np.cos(theta[j+1]) / lam
        
        # 计算p值
        if polarization == 'TE':
            p = n_layer[j+1] * np.cos(theta[j+1])
        else:  # TM
            p = np.cos(theta[j+1]) / n_layer[j+1]
        
        # 构建单层传输矩阵
        M_j = np.array([
            [np.cos(delta), 1j * np.sin(delta) / p],
            [1j * p * np.sin(delta), np.cos(delta)]
        ])
        
        # 矩阵乘法
        M = M @ M_j
    
    return M

def reflectance_transmittance(n_layer, d_layer, theta0, lam, polarization='TE'):
    """
    计算反射率和透射率
    
    参数:
        同 transfer_matrix
    返回:
        R, T: 反射率和透射率
    """
    M = transfer_matrix(n_layer, d_layer, theta0, lam, polarization)
    
    # 计算p0和p_last
    if polarization == 'TE':
        p0 = n_layer[0] * np.cos(theta0)
        p_last = n_layer[-1] * np.cos(np.arcsin(n_layer[0] * np.sin(theta0) / n_layer[-1]))
    else:
        p0 = np.cos(theta0) / n_layer[0]
        p_last = np.cos(np.arcsin(n_layer[0] * np.sin(theta0) / n_layer[-1])) / n_layer[-1]
    
    # 计算反射系数和透射系数
    r = (M[0,0]*p0 + M[0,1]*p0*p_last - M[1,0] - M[1,1]*p_last) / \
        (M[0,0]*p0 + M[0,1]*p0*p_last + M[1,0] + M[1,1]*p_last)
    
    t = 2 * p0 / (M[0,0]*p0 + M[0,1]*p0*p_last + M[1,0] + M[1,1]*p_last)
    
    R = np.abs(r)**2
    T = np.abs(t)**2 * p_last / p0
    
    return R, T

# 示例：计算单层MgF2增透膜的反射谱
if __name__ == "__main__":
    # 参数设置
    n_air = 1.0
    n_film = 1.38  # MgF2
    n_sub = 1.5     # 玻璃
    d_film = 100    # nm
    
    wavelengths = np.linspace(400, 700, 300)
    R_vals = []
    
    for lam in wavelengths:
        R, _ = reflectance_transmittance(
            [n_air, n_film, n_sub],
            [d_film],
            0,  # 正入射
            lam,
            'TE'
        )
        R_vals.append(R)
    
    # 绘制结果
    plt.figure(figsize=(10, 6))
    plt.plot(wavelengths, R_vals, 'b-', linewidth=2)
    plt.xlabel('Wavelength (nm)')
    plt.ylabel('Reflectance')
    plt.title('Single-layer MgF2 Anti-reflection Coating')
    plt.grid(True)
    plt.show()
```

## 应用场景

### 1. 抗反射涂层

抗反射涂层是最常见的光学薄膜应用。通过沉积一层或多层折射率介于空气和基底之间的薄膜，可以显著降低表面反射。例如，相机镜头上的MgF₂单层增透膜（折射率≈1.38）在可见光中心波长处可将反射率从4%降至约1%。

### 2. 高反射镜

通过交替沉积高折射率（如TiO₂，n≈2.4）和低折射率（如SiO₂，n≈1.46）材料，可以制备宽带高反射镜。这种分布式布拉格反射镜（DBR）在激光谐振腔、光纤通信等领域有重要应用。通过传输矩阵法可以精确设计反射带的中心波长和带宽。
