# 扩散模型原理：从噪声中重构数据之美

## 引言

在生成模型领域，GAN、VAE和Flow-based模型各领风骚，但近年来，扩散模型（Diffusion Models）凭借其稳定的训练过程和卓越的生成质量，迅速成为图像生成、音频合成等任务的核心技术，DALL·E 2和Stable Diffusion等知名系统均基于此。扩散模型的核心思想借鉴了热力学中的扩散过程：通过逐步向数据中添加噪声，将其破坏为纯噪声；然后学习一个反向过程，从噪声中逐步恢复出原始数据。本文将深入剖析扩散模型的数学原理，并提供完整的代码实现，帮助你理解这一强大技术的内在机制。

## 核心概念

### 1. 前向扩散过程

前向扩散过程是一个马尔可夫链，它逐步向数据 $x_0$ 中添加高斯噪声，经过 $T$ 步后，数据变为纯高斯噪声 $x_T$。每一步的转移可以表示为：

$$q(x_t | x_{t-1}) = \mathcal{N}(x_t; \sqrt{1-\beta_t} x_{t-1}, \beta_t \mathbf{I})$$

其中 $\beta_t$ 是噪声调度表（variance schedule），控制了每一步添加噪声的强度。通过重参数化技巧，我们可以直接从 $x_0$ 得到任意时刻 $t$ 的 $x_t$：

$$x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1-\bar{\alpha}_t} \epsilon, \quad \epsilon \sim \mathcal{N}(0, \mathbf{I})$$

这里 $\alpha_t = 1 - \beta_t$，$\bar{\alpha}_t = \prod_{i=1}^t \alpha_i$。

### 2. 反向去噪过程

反向过程的目标是学习一个神经网络 $p_\theta(x_{t-1} | x_t)$ 来逆转前向扩散。由于当 $\beta_t$ 足够小时，反向过程也可以近似为高斯分布，因此我们建模：

$$p_\theta(x_{t-1} | x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t))$$

在实践中，通常固定方差 $\Sigma_\theta = \sigma_t^2 \mathbf{I}$（其中 $\sigma_t^2 = \beta_t$ 或 $\tilde{\beta}_t$），只学习均值 $\mu_\theta$。

### 3. 变分下界 (ELBO)

扩散模型的训练目标是最小化负对数似然的变分下界（ELBO）。通过推导，ELBO可以简化为一个简单的均方误差损失：

$$L_{\text{simple}} = \mathbb{E}_{t, x_0, \epsilon} \left[ \|\epsilon - \epsilon_\theta(x_t, t)\|^2 \right]$$

这意味着神经网络 $\epsilon_\theta$ 的任务是预测添加到 $x_0$ 上的噪声 $\epsilon$。

## 数学原理

### 关键公式推导

**前向过程**的封闭形式（从 $x_0$ 到 $x_t$）：

$$q(x_t | x_0) = \mathcal{N}(x_t; \sqrt{\bar{\alpha}_t} x_0, (1-\bar{\alpha}_t) \mathbf{I})$$

**反向过程**的均值可以通过贝叶斯公式推导得到。给定 $x_t$ 和 $x_0$，后验分布 $q(x_{t-1} | x_t, x_0)$ 是高斯分布：

$$q(x_{t-1} | x_t, x_0) = \mathcal{N}(x_{t-1}; \tilde{\mu}_t(x_t, x_0), \tilde{\beta}_t \mathbf{I})$$

其中：

$$\tilde{\mu}_t(x_t, x_0) = \frac{\sqrt{\bar{\alpha}_{t-1}} \beta_t}{1-\bar{\alpha}_t} x_0 + \frac{\sqrt{\alpha_t}(1-\bar{\alpha}_{t-1})}{1-\bar{\alpha}_t} x_t$$

$$\tilde{\beta}_t = \frac{1-\bar{\alpha}_{t-1}}{1-\bar{\alpha}_t} \beta_t$$

由于训练时我们让网络预测噪声 $\epsilon$，而 $x_0 = \frac{x_t - \sqrt{1-\bar{\alpha}_t} \epsilon}{\sqrt{\bar{\alpha}_t}}$，代入上式得到：

$$\mu_\theta(x_t, t) = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1-\bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right)$$

**采样过程**的迭代公式：

$$x_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1-\bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right) + \sigma_t z, \quad z \sim \mathcal{N}(0, \mathbf{I})$$

当 $t=1$ 时，$z=0$。

## 代码实现

以下是一个完整的扩散模型训练与采样实现，使用PyTorch：

```python
import torch
import torch.nn as nn
import numpy as np
from torch.utils.data import DataLoader, TensorDataset

class SimpleDiffusion:
    def __init__(self, T=1000, beta_start=1e-4, beta_end=0.02):
        self.T = T
        # 线性噪声调度表
        self.betas = torch.linspace(beta_start, beta_end, T)
        self.alphas = 1. - self.betas
        self.alpha_bars = torch.cumprod(self.alphas, dim=0)
    
    def q_sample(self, x0, t, noise=None):
        """前向过程：从x0得到xt"""
        if noise is None:
            noise = torch.randn_like(x0)
        sqrt_alpha_bar = torch.sqrt(self.alpha_bars[t])[:, None, None, None]
        sqrt_one_minus_alpha_bar = torch.sqrt(1. - self.alpha_bars[t])[:, None, None, None]
        return sqrt_alpha_bar * x0 + sqrt_one_minus_alpha_bar * noise, noise
    
    def p_sample(self, model, xt, t):
        """反向过程单步采样"""
        t_batch = t.expand(xt.shape[0]).to(xt.device)
        # 预测噪声
        predicted_noise = model(xt, t_batch.float() / self.T)
        # 计算均值
        alpha_t = self.alphas[t][:, None, None, None].to(xt.device)
        beta_t = self.betas[t][:, None, None, None].to(xt.device)
        alpha_bar_t = self.alpha_bars[t][:, None, None, None].to(xt.device)
        
        mean = (1 / torch.sqrt(alpha_t)) * (xt - (beta_t / torch.sqrt(1 - alpha_bar_t)) * predicted_noise)
        
        # 添加噪声（最后一步不加）
        if t[0] > 0:
            noise = torch.randn_like(xt)
            sigma_t = torch.sqrt(beta_t)
            return mean + sigma_t * noise
        else:
            return mean

class SimpleUNet(nn.Module):
    """简化的U-Net用于噪声预测"""
    def __init__(self):
        super().__init__()
        self.time_embed = nn.Sequential(
            nn.Linear(1, 128),
            nn.ReLU(),
            nn.Linear(128, 128)
        )
        self.encoder = nn.Sequential(
            nn.Conv2d(1, 64, 3, padding=1),
            nn.ReLU(),
            nn.Conv2d(64, 64, 3, padding=1),
            nn.ReLU(),
            nn.Conv2d(64, 128, 3, stride=2, padding=1),
            nn.ReLU()
        )
        self.decoder = nn.Sequential(
            nn.ConvTranspose2d(128, 64, 4, stride=2, padding=1),
            nn.ReLU(),
            nn.Conv2d(64, 64, 3, padding=1),
            nn.ReLU(),
            nn.Conv2d(64, 1, 3, padding=1)
        )
    
    def forward(self, x, t):
        t_emb = self.time_embed(t.unsqueeze(-1))
        # 将时间嵌入广播到空间维度
        t_emb = t_emb[:, :, None, None].expand(-1, -1, x.shape[2], x.shape[3])
        x = self.encoder(x)
        x = torch.cat([x, t_emb], dim=1)
        x = self.decoder(x)
        return x

# 训练示例
def train_diffusion():
    # 生成简单数据集（8x8的圆圈图案）
    dataset = []
    for _ in range(1000):
        img = torch.zeros(1, 8, 8)
        cx, cy = np.random.randint(2, 6, 2)
        for i in range(8):
            for j in range(8):
                if (i-cx)**2 + (j-cy)**2 <= 4:
                    img[0, i, j] = 1.0
        dataset.append(img)
    dataset = torch.stack(dataset)
    
    loader = DataLoader(TensorDataset(dataset), batch_size=64, shuffle=True)
    model = SimpleUNet()
    diffusion = SimpleDiffusion(T=100)
    optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)
    
    for epoch in range(50):
        for batch, in loader:
            optimizer.zero_grad()
            t = torch.randint(0, diffusion.T, (batch.shape[0],))
            xt, noise = diffusion.q_sample(batch, t)
            predicted_noise = model(xt, t.float() / diffusion.T)
            loss = nn.MSELoss()(predicted_noise, noise)
            loss.backward()
            optimizer.step()
        print(f"Epoch {epoch}, Loss: {loss.item():.4f}")
    
    # 采样
    model.eval()
    with torch.no_grad():
        x = torch.randn(1, 1, 8, 8)
        for t in reversed(range(diffusion.T)):
            x = diffusion.p_sample(model, x, torch.tensor([t]))
    return x
```

## 应用场景

### 1. 图像生成

扩散模型最著名的应用是图像生成。Stable Diffusion 通过将扩散过程在潜在空间中执行（Latent Diffusion），大幅降低了计算成本，使得在消费级GPU上也能生成高质量图像。模型可以从文本描述生成照片级真实的图像，广泛应用于创意设计、广告等领域。

### 2. 音频合成

在语音和音乐生成中，扩散模型同样表现出色。例如，WaveGrad 和 DiffWave 将扩散模型应用于音频波形生成，能够产生高质量的自然语音。与自回归模型相比，扩散模型可以并行生成，大大提高了推理速度。

## 总结

扩散模型通过模拟热力学中的扩散过程，构建了一个从数据到噪声再到数据的完整生成框架。其核心要点包括：

1. **前向过程**：通过马尔可夫链逐步添加噪声，最终将数据变为纯噪声
2. **反向过程**：学习神经网络预测噪声，从噪声中逐步恢复数据
3. **训练目标**：简化为预测添加噪声的均方误差损失
4. **采样过程**：通过迭代去噪，从随机噪声生成新样本

扩散模型的成功不仅在于其生成质量，更在于其训练稳定性和理论优雅性。随着采样加速技术（如DDIM、LCM）的发展，扩散模型正在成为生成式AI的基石技术。理解其原理，将为你打开探索更高级生成模型的大门。
