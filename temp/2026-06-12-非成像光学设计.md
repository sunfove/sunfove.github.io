# 非成像光学设计：从光学扩展量守恒到CPC聚光器

## 引言

在传统成像光学中，我们追求的是物与像之间完美的几何对应关系；然而在太阳能收集、LED照明等应用场景中，我们并不关心“像”的质量，只希望尽可能高效地将光能从光源传输到目标区域。非成像光学正是为此而生。它不追求成像质量，而是关注光能的传输效率与均匀性。其核心思想——光学扩展量守恒与边缘光线原理——为设计高效聚光器、匀光器等系统提供了坚实的理论基础。本文将深入剖析这些核心概念，并通过复合抛物面聚光器（CPC）的数学推导与Python实现，展示非成像光学的实际应用价值。

## 核心概念

### 光学扩展量（Etendue）

光学扩展量是描述光束在空间和角度上“占据”程度的物理量，是光学系统的基本不变量。对于三维系统，其定义为：

$$ E = n^2 \iint \cos\theta \, dA \, d\Omega $$

其中，$n$ 为介质折射率，$dA$ 是面积微元，$d\Omega$ 是立体角微元，$\theta$ 是光线方向与面元法线的夹角。在理想光学系统中，光学扩展量守恒——这意味着我们无法在不损失能量的情况下同时缩小光束的截面积和发散角。

对于二维系统（如线性聚光器），光学扩展量简化为：

$$ E_{2D} = n \int \cos\theta \, dx \, d\theta $$

理解光学扩展量守恒是设计非成像系统的关键：它给出了理论上最大的聚光比。

### 边缘光线原理

边缘光线原理是非成像光学设计的基石：如果从光源边缘发出的所有光线都能到达目标区域的边缘，那么从光源内部发出的光线也必然能到达目标区域内部。这极大地简化了设计问题——我们只需要处理边界光线，而非所有光线。

### 复合抛物面聚光器（CPC）

CPC是一种典型的非成像聚光器，由两个旋转抛物面组成，能够实现理论极限的聚光比。其核心思想是将光源边缘的光线引导至接收器的边缘，从而最大程度地收集光能。二维CPC的轮廓由抛物线方程决定，其设计参数包括接收半角 $\theta_a$ 和接收器宽度 $2a$。

## 数学原理

### 光学扩展量的严格定义

考虑一个光束通过面积为 $A$ 的截面，光线方向与法线夹角为 $\theta$，在立体角 $\Omega$ 内传播，光学扩展量为：

$$ E = n^2 \int_A \int_\Omega \cos\theta \, d\Omega \, dA $$

对于二维情况，设截面长度为 $x$，角度范围为 $\theta$，则：

$$ E_{2D} = n \int_x \int_\theta \cos\theta \, d\theta \, dx $$

### 二维CPC轮廓方程

设计一个接收半角为 $\theta_a$，接收器宽度为 $2a$ 的二维CPC。其轮廓由两个对称的抛物线弧段组成，每个抛物线以接收器边缘为焦点，且轴线与系统光轴成 $\theta_a$ 角。

以右侧抛物线为例，设接收器右边缘位于 $(a, 0)$，抛物线轴线方向与 $x$ 轴夹角为 $\theta_a$。抛物线参数方程可写为：

$$ x(t) = a + \frac{2f \cos\theta_a \cdot t - t^2 \sin\theta_a}{1 - \sin\theta_a} $$
$$ y(t) = \frac{t^2 \cos\theta_a + 2f \sin\theta_a \cdot t}{1 - \sin\theta_a} $$

其中 $f = a(1 + \sin\theta_a)$ 为焦距，$t$ 为参数（从0到某个最大值）。更常见的简化形式是使用极坐标或直接给出显式方程。对于二维CPC，其轮廓方程可表达为：

$$ r(\phi) = \frac{2f}{1 + \sin(\phi - \theta_a)} $$

其中 $r$ 是从焦点到轮廓上点的距离，$\phi$ 是极角。当 $\phi$ 从 $\theta_a$ 变化到 $\pi/2$ 时，描出完整的抛物线弧段。

## 代码实现

以下Python代码实现了一个二维CPC轮廓的生成与可视化：

```python
import numpy as np
import matplotlib.pyplot as plt

def generate_cpc_profile(a, theta_a, num_points=1000):
    """
    生成二维CPC右侧轮廓
    
    参数:
        a: 接收器半宽度
        theta_a: 接收半角（弧度）
        num_points: 轮廓点数
    
    返回:
        x, y: 轮廓坐标数组
    """
    f = a * (1 + np.sin(theta_a))  # 焦距
    
    # 参数t从0到最大值，确保轮廓延伸到所需高度
    t_max = 2 * f / (1 - np.sin(theta_a))  # 最大t值
    t = np.linspace(0, t_max, num_points)
    
    # 计算轮廓坐标
    denom = 1 - np.sin(theta_a)
    x = a + (2 * f * np.cos(theta_a) * t - t**2 * np.sin(theta_a)) / denom
    y = (t**2 * np.cos(theta_a) + 2 * f * np.sin(theta_a) * t) / denom
    
    return x, y

# 参数设置
a = 10.0  # 接收器半宽度（mm）
theta_a = np.radians(30)  # 接收半角30度

# 生成右侧轮廓
x_right, y_right = generate_cpc_profile(a, theta_a)

# 通过对称生成左侧轮廓
x_left = -x_right
y_left = y_right

# 绘制CPC轮廓
plt.figure(figsize=(10, 6))
plt.plot(x_right, y_right, 'b-', linewidth=2, label='右侧轮廓')
plt.plot(x_left, y_left, 'b-', linewidth=2, label='左侧轮廓')

# 绘制接收器
plt.plot([-a, a], [0, 0], 'r-', linewidth=4, label='接收器')

# 绘制边缘光线示意
# 从接收器右边缘发出的边缘光线
angle = np.linspace(0, theta_a, 5)
for ang in angle:
    x_ray = a + np.array([0, 50]) * np.cos(ang)
    y_ray = 0 + np.array([0, 50]) * np.sin(ang)
    plt.plot(x_ray, y_ray, 'g--', alpha=0.3)

plt.xlabel('x (mm)')
plt.ylabel('y (mm)')
plt.title(f'二维CPC轮廓 (a={a}mm, θ_a={np.degrees(theta_a):.0f}°)')
plt.axis('equal')
plt.grid(True, alpha=0.3)
plt.legend()
plt.xlim(-50, 50)
plt.ylim(0, 60)
plt.show()

# 计算理论聚光比
C_theoretical = 1 / np.sin(theta_a)
print(f"理论聚光比: {C_theoretical:.2f}")
print(f"接收器宽度: {2*a:.1f} mm")
print(f"CPC开口宽度: {2 * x_right[-1]:.1f} mm")
```

运行这段代码，你将看到CPC的轮廓曲线以及从接收器边缘发出的边缘光线。理论聚光比 $C = 1/\sin\theta_a$ 会在控制台输出。

## 应用场景

### 太阳能聚光器

CPC最经典的应用是太阳能集热器。将CPC的接收器替换为太阳能吸收管，开口朝向太阳，可以在一天中无需跟踪系统即可收集阳光。由于CPC的接收半角设计，即使太阳位置有±θ_a的变化，所有光线仍能被有效收集。例如，设计θ_a=30°的CPC，理论上可实现2倍的聚光比，大大提高了集热效率。

### LED准直与均匀照明

在LED照明中，常需要将LED发出的朗伯体光分布转换为均匀的平行光束。利用光学扩展量守恒原理，可以设计CPC或类似结构作为准直器。将LED置于CPC的接收器位置（小端），CPC的大端输出近似平行的光束。通过优化轮廓，可以在保持高光效的同时实现均匀的照度分布，广泛应用于汽车前照灯、投影仪等系统。

## 总结

非成像光学设计以光学扩展量守恒为核心，通过边缘光线原理简化了系统设计。本文从基础概念出发，推导了二维CPC的轮廓方程，并提供了可运行的Python实现代码。关键要点回顾：

1. **光学扩展量守恒**：决定了系统理论上的最大聚光比，是设计的根本约束
2. **边缘光线原理**：将复杂的光线追踪问题简化为边界处理
3. **CPC设计**：通过抛物线轮廓实现理论极限聚光，方程简洁且易于实现
4. **实际应用**：从太阳能收集到LED照明，非成像光学正在改变我们的能源与照明技术

非成像光学为我们提供了一种全新的视角：有时候，不追求“完美成像”，反而能实现更高效的能量传递。这正是工程智慧的魅力所在。
