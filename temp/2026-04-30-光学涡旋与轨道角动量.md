# 光学涡旋与轨道角动量：光的“旋转”秘密

## 1. 引言

光，作为电磁波，除了我们熟知的能量和动量，还隐藏着一个奇妙的“旋转”属性——轨道角动量（OAM）。当一束光携带着螺旋形的波前，它便形成了一个光学涡旋（Optical Vortex）。这不仅是物理学的有趣现象，更在光通信、超分辨成像、光镊操控等领域展现出革命性的潜力。想象一下，一束光像“微型龙卷风”一样旋转，携带的不仅仅是能量，还有巨大的信息容量——这正是光学涡旋的魅力所在。本文将带你深入这一领域的核心概念、数学原理，并通过代码亲手生成一个光学涡旋。

## 2. 核心概念

### 2.1 光学涡旋 (Optical Vortex)
光学涡旋是光场中一种相位奇异点。在这一点上，光强为零（即暗核），而相位围绕该点呈螺旋状变化。这种螺旋相位结构使得光波前像“开瓶器”一样旋转。

### 2.2 轨道角动量 (OAM)
与自旋角动量（对应圆偏振）不同，OAM 描述的是光波前整体旋转带来的角动量。它由相位奇点的拓扑结构决定，每个光子携带的 OAM 为 \( l\hbar \)，其中 \( l \) 是整数。

### 2.3 拓扑荷数 (Topological Charge, \( l \))
拓扑荷数 \( l \) 是描述光学涡旋“旋转圈数”的关键参数。它定义为相位绕奇点一周的变化量除以 \( 2\pi \)：
\[
l = \frac{1}{2\pi} \oint \nabla \phi \cdot d\mathbf{l}
\]
其中 \( \phi \) 是光场的相位。\( l \) 为正时，波前右旋（顺时针）；为负时左旋（逆时针）。\( |l| \) 越大，涡旋的“扭转”越剧烈。

## 3. 数学原理

### 3.1 拉盖尔-高斯 (LG) 光束
光学涡旋最典型的载体是拉盖尔-高斯光束，其电场分布的数学表达式为：
\[
E_{p,l}(r, \phi, z) = \frac{C_{p,l}}{w(z)} \left( \frac{\sqrt{2}r}{w(z)} \right)^{|l|} L_p^{|l|} \left( \frac{2r^2}{w^2(z)} \right) \exp\left( -\frac{r^2}{w^2(z)} \right) \exp\left( -i k z \right) \exp\left( i l \phi \right) \exp\left( i \Phi_{Gouy}(z) \right)
\]
其中：
- \( r, \phi \)：柱坐标下的径向和角向坐标
- \( w(z) \)：光束半径，\( w(z) = w_0 \sqrt{1 + (z/z_R)^2} \)，\( z_R \) 为瑞利长度
- \( L_p^{|l|} \)：广义拉盖尔多项式
- \( \Phi_{Gouy}(z) \)：古依相位
- 关键的螺旋相位项：\( \exp(i l \phi) \)

### 3.2 OAM 密度与光强的关系
从经典电磁理论，OAM 密度 \( \mathbf{j} \) 与坡印廷矢量关系密切。对于沿 z 轴传播的 LG 光束，其 z 方向 OAM 密度可简化为：
\[
j_z \propto \frac{l}{\omega} |E|^2
\]
其中 \( \omega \) 是角频率。这表明：OAM 密度正比于光强 \( |E|^2 \) 和拓扑荷数 \( l \)。光强越强，OAM 密度越大；拓扑荷数越高，单个光子携带的角动量越大。

从更直观的角度，OAM 通量可表示为：
\[
\mathbf{L} = \frac{1}{2\pi} \iint \mathbf{r} \times \langle \mathbf{S} \rangle dA
\]
其中 \( \langle \mathbf{S} \rangle \) 是时间平均坡印廷矢量。对于 LG 光束，计算可得 \( L_z = l \hbar N \)，\( N \) 为光子数。

## 4. 代码实现

下面我们用 Python 生成一个拓扑荷数 \( l = 1 \) 的 LG 光束，并可视化其强度和相位分布。

```python
import numpy as np
import matplotlib.pyplot as plt

def lg_beam(r, phi, l, p, w0, z, wavelength):
    """
    计算拉盖尔-高斯光束的复振幅
    参数:
        r, phi: 极坐标网格
        l: 拓扑荷数
        p: 径向指数（这里取 p=0 简化）
        w0: 束腰半径
        z: 传播距离
        wavelength: 波长
    返回:
        E: 复电场分布
    """
    # 波数
    k = 2 * np.pi / wavelength
    # 瑞利长度
    zR = np.pi * w0**2 / wavelength
    # 光束半径
    w_z = w0 * np.sqrt(1 + (z / zR)**2)
    # 古依相位（p=0时）
    gouy_phase = (2*p + abs(l) + 1) * np.arctan(z / zR)
    
    # 归一化常数（简化版）
    C = np.sqrt(2 / (np.pi * w_z**2)) * np.sqrt(np.math.factorial(p) / np.math.factorial(p + abs(l)))
    
    # 径向部分（p=0时，L_0^|l| = 1）
    radial = (np.sqrt(2) * r / w_z)**abs(l) * np.exp(-r**2 / w_z**2)
    
    # 复振幅
    E = C * radial * np.exp(1j * l * phi) * np.exp(-1j * k * z) * np.exp(1j * gouy_phase)
    return E

# 参数设置
wavelength = 532e-9  # 532 nm
w0 = 1e-3  # 束腰 1 mm
z = 0  # 在束腰平面
l = 1  # 拓扑荷数
p = 0  # 径向指数

# 创建网格
N = 512
x = np.linspace(-2e-3, 2e-3, N)
y = np.linspace(-2e-3, 2e-3, N)
X, Y = np.meshgrid(x, y)
r = np.sqrt(X**2 + Y**2)
phi = np.arctan2(Y, X)

# 计算电场
E = lg_beam(r, phi, l, p, w0, z, wavelength)

# 强度和相位
I = np.abs(E)**2
phase = np.angle(E)

# 绘图
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

# 强度图
im1 = ax1.imshow(I, extent=[-2, 2, -2, 2], cmap='hot')
ax1.set_title(f'强度分布 (l={l})')
ax1.set_xlabel('x (mm)')
ax1.set_ylabel('y (mm)')
plt.colorbar(im1, ax=ax1)

# 相位图
im2 = ax2.imshow(phase, extent=[-2, 2, -2, 2], cmap='hsv', vmin=-np.pi, vmax=np.pi)
ax2.set_title(f'相位分布 (l={l})')
ax2.set_xlabel('x (mm)')
ax2.set_ylabel('y (mm)')
plt.colorbar(im2, ax=ax2)

plt.tight_layout()
plt.show()
```

**代码说明**:
- 我们使用极坐标网格计算 LG 光束的复振幅
- 强度图显示中心暗核（光学涡旋的特征）
- 相位图呈现螺旋结构，从 0 到 \( 2\pi \) 变化一次（对应 \( l=1 \)）

## 5. 应用场景

### 5.1 高容量光通信
OAM 模式具有正交性，不同拓扑荷数的光束互不干扰。理论上，可以将不同 \( l \) 值的光束叠加在同一光纤中，实现空分复用，极大提升通信容量。例如，使用 \( l = \pm 1, \pm 2, \pm 3 \) 等模式，可将单根光纤的传输容量提升数十倍。

### 5.2 光镊与微粒操控
光学涡旋的暗核可以“捕获”微粒，而 OAM 可以驱动微粒旋转。这种“光学扳手”在生物医学中用于操控细胞、细菌等微小物体。例如，使用 \( l=5 \) 的高阶涡旋，可以以可控速度旋转微米级粒子，实现微流控芯片中的“光学搅拌器”。

## 6. 总结

本文深入探讨了光学涡旋与轨道角动量的核心概念、数学原理和实际应用。关键要点如下：

1. **光学涡旋**：具有螺旋相位结构的光场，中心为暗核。
2. **拓扑荷数 \( l \)**：描述涡旋的“旋转圈数”，决定 OAM 大小。
3. **LG 光束**：OAM 的典型载体，其电场分布由拉盖尔多项式、高斯包络和螺旋相位共同决定。
4. **OAM 密度**：正比于光强和拓扑荷数，物理上对应光波前的整体旋转。
5. **应用前景**：从光通信到微粒操控，OAM 正在重塑光学技术的未来。

光学涡旋的研究仍在快速发展，随着超表面和光纤技术的发展，OAM 有望在量子信息、超分辨成像等领域发挥更大作用。希望本文能为你打开一扇通往“旋转光”世界的大门。
