# 微纳光学与超表面：驾驭光的“魔法平面”

## 1. 引言

想象一下，如果一片比头发丝还薄数百倍的平面，能够像透镜一样聚焦光线，像棱镜一样偏折光束，甚至能生成全息图像——这不是科幻，而是微纳光学中最令人兴奋的前沿领域：**超表面**。传统光学器件依赖光在传播路径中逐渐积累的相位变化，这要求透镜必须具有弯曲的曲面，体积难以缩小。而超表面通过亚波长厚度的纳米结构阵列，在平面内直接引入突变相位，实现了对光波的任意操控。这一突破不仅让光学系统走向“扁平化”，更开启了集成光子学、增强现实、超分辨成像等领域的全新可能。本文将带你深入超表面的物理本质、数学原理，并通过代码实践其设计方法。

## 2. 核心概念

### 2.1 超表面 (Metasurface)
超表面是一种由亚波长尺度（通常为λ/10 ~ λ/5）的纳米天线或谐振器构成的二维平面结构。每个“天线”单元（称为元原子）可以独立设计其几何参数（如形状、尺寸、朝向），从而对入射光的相位、振幅和偏振进行局部调控。与传统渐变折射率器件不同，超表面在界面处引入**突变的相位跳变**，使得光波前可以被“编程”成任意形状。

### 2.2 广义斯涅耳定律 (Generalized Snell's Law)
这是超表面的理论基础。传统斯涅耳定律假设界面两侧介质均匀，相位连续。但当界面引入一个空间变化的相位突变Φ(x)时，光线的偏折规律被改写。对于折射情况，广义斯涅耳定律为：
$$ n_t \sin\theta_t - n_i \sin\theta_i = \frac{\lambda_0}{2\pi} \frac{d\Phi}{dx} $$

其中，\(n_i\)和\(n_t\)是入射和透射介质的折射率，\(\theta_i\)和\(\theta_t\)是入射角和透射角，\(\lambda_0\)是真空波长。\(\frac{d\Phi}{dx}\)是界面处的相位梯度。这个公式告诉我们：通过设计相位梯度，我们可以使光线向任意方向偏折，甚至实现反常折射（如负折射）。

### 2.3 等离激元 (Plasmonics)
等离激元是金属中自由电子集体振荡与电磁场耦合形成的准粒子。在超表面中，金属纳米结构（如金、银）可以支持**局域表面等离激元共振**（LSPR）。当入射光频率接近纳米结构的本征共振频率时，局域电场被极大增强，同时伴随强烈的相位变化（0到π）。LSPR的共振频率强烈依赖于纳米结构的几何形状和周围介电环境，这为相位调控提供了物理机制。

## 3. 数学原理

### 3.1 广义斯涅耳定律的推导

考虑一束平面波从介质1（折射率\(n_i\)）入射到介质2（折射率\(n_t\)）的界面。设界面位于z=0平面，且引入一个随x变化的相位突变Φ(x)。根据费马原理，光从A点（\(x_A, z_A\)）到B点（\(x_B, z_B\)）的实际路径应使总相位取极值。总相位包括传播相位和界面突变相位：
$$ \phi_{\text{total}} = k_0 n_i \sqrt{(x-x_A)^2 + z_A^2} + \Phi(x) + k_0 n_t \sqrt{(x_B-x)^2 + z_B^2} $$

其中\(k_0 = 2\pi/\lambda_0\)。对x求导并令导数为零：
$$ \frac{d\phi_{\text{total}}}{dx} = k_0 n_i \frac{x-x_A}{\sqrt{(x-x_A)^2 + z_A^2}} + \frac{d\Phi}{dx} - k_0 n_t \frac{x_B-x}{\sqrt{(x_B-x)^2 + z_B^2}} = 0 $$

注意到\(\sin\theta_i = (x-x_A)/d_i\)，\(\sin\theta_t = (x_B-x)/d_t\)，其中\(d_i, d_t\)是光线路径长度。代入并整理即得：
$$ n_t \sin\theta_t - n_i \sin\theta_i = \frac{\lambda_0}{2\pi} \frac{d\Phi}{dx} $$

### 3.2 局域表面等离激元共振频率

对于球形纳米颗粒（半径a << λ），在准静态近似下，其极化率由Clausius-Mossotti关系给出。共振发生在介电函数实部满足：
$$ \text{Re}[\epsilon_m(\omega)] = -2\epsilon_d $$

其中\(\epsilon_m\)是金属的介电函数（通常用Drude模型\(\epsilon_m(\omega) = \epsilon_\infty - \omega_p^2/(\omega^2 + i\gamma\omega)\)），\(\epsilon_d\)是周围介质的介电常数。由此可解出共振频率：
$$ \omega_{\text{LSPR}} = \frac{\omega_p}{\sqrt{\epsilon_\infty + 2\epsilon_d}} $$

对于非球形结构，共振频率会偏移，且会出现多极共振模式。

## 4. 代码实现

下面我们用Python模拟一个简单的超表面设计：通过调整金纳米棒的尺寸，实现从0到2π的相位覆盖，并验证广义斯涅耳定律。这里使用Mie理论（简化版）计算单个纳米棒的散射相位。

```python
import numpy as np
import matplotlib.pyplot as plt

# 物理常数
c = 3e8  # 光速 m/s
lambda0 = 633e-9  # 工作波长 633nm (He-Ne激光)
omega0 = 2 * np.pi * c / lambda0  # 角频率

# 金 (Au) 的Drude参数 (Johnson & Christy)
epsilon_inf = 9.84
omega_p = 1.36e16  # 等离子体频率 rad/s
gamma = 1.05e14   # 阻尼系数 rad/s

def gold_permittivity(omega):
    """计算金的介电函数 (Drude模型)"""
    return epsilon_inf - omega_p**2 / (omega**2 + 1j * gamma * omega)

def phase_from_nanorod(L, w, h=40e-9):
    """
    计算矩形金纳米棒的散射相位 (简化模型)
    基于偶极近似，共振频率随长宽比变化
    参数:
        L: 长度 (m)
        w: 宽度 (m)
        h: 高度 (m) 固定
    返回:
        散射相位 (rad)
    """
    # 有效体积
    V = L * w * h
    # 退极化因子 (近似，对于长椭球)
    # 这里简化为长宽比的函数
    aspect = L / w
    # 共振频率偏移 (经验公式)
    omega_res = omega0 / (1 + 0.1 * (aspect - 2))  # 假设aspect=2时共振于工作波长
    
    # 金属介电函数在共振频率处
    eps_m = gold_permittivity(omega_res)
    eps_d = 1.0  # 空气
    
    # 极化率 (准静态近似)
    alpha = V * (eps_m - eps_d) / (eps_d + 3 * (eps_m - eps_d) * 0.3)  # 简化因子
    
    # 散射相位 = arctan(Im(alpha)/Re(alpha))
    phase = np.arctan2(alpha.imag, alpha.real)
    return phase

# 设计超表面：沿x方向排列不同长度的纳米棒
N = 20  # 单元数
x_pos = np.linspace(0, 10*lambda0, N)  # 空间位置
L_values = np.linspace(80e-9, 200e-9, N)  # 长度从80nm到200nm
w = 50e-9  # 固定宽度

phases = np.array([phase_from_nanorod(L, w) for L in L_values])

# 归一化相位到 [0, 2π)
phases = (phases - phases.min()) / (phases.max() - phases.min()) * 2 * np.pi

# 计算相位梯度
dPhi_dx = np.gradient(phases, x_pos)

# 预测偏折角 (广义斯涅耳定律，假设垂直入射)
n_i = 1.0  # 空气
n_t = 1.0  # 透射到空气
theta_t = np.arcsin(lambda0 / (2*np.pi) * dPhi_dx)

# 绘图
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 8))

ax1.plot(x_pos*1e6, phases, 'o-', label='相位分布')
ax1.set_xlabel('x (μm)')
ax1.set_ylabel('相位 (rad)')
ax1.set_title('超表面相位分布')
ax1.legend()
ax1.grid(True)

ax2.plot(x_pos*1e6, np.degrees(theta_t), 's-', color='red', label='偏折角')
ax2.axhline(0, color='gray', linestyle='--')
ax2.set_xlabel('x (μm)')
ax2.set_ylabel('偏折角 (度)')
ax2.set_title('广义斯涅耳定律预测的偏折角')
ax2.legend()
ax2.grid(True)

plt.tight_layout()
plt.show()

print(f"平均偏折角: {np.mean(np.degrees(theta_t)):.2f}°")
```

**代码说明**：
- 使用Drude模型模拟金的介电函数
- 通过改变纳米棒长度（80-200nm）实现相位调控
- 计算相位梯度后，用广义斯涅耳定律预测偏折角
- 运行结果将显示相位近似线性变化，导致光线偏向一个方向

## 5. 应用场景

### 5.1 超薄平面透镜 (Metalens)
传统透镜依靠曲面折射，厚度至少数毫米。而超表面透镜通过纳米柱阵列在平面内实现双曲相位分布：
$$ \Phi(r) = -\frac{2\pi}{\lambda} \left( \sqrt{r^2 + f^2} - f \right) $$
其中f是焦距。这种透镜厚度仅数百纳米，可实现衍射极限聚焦，已用于微型相机、内窥镜等场景。

### 5.2 全息显示与光学隐身
通过设计超表面的相位与振幅分布，可以生成任意波前。例如，在增强现实眼镜中，超表面可将图像直接投射到视网膜；在隐身斗篷中，超表面可引导光线绕过物体，实现“光学隐形”。2015年，哈佛大学Capasso团队展示了基于超表面的宽带隐身地毯。

## 6. 总结

超表面是微纳光学领域的一场革命。本文我们深入探讨了：
1. **核心概念**：超表面通过亚波长结构在界面引入突变相位，广义斯涅耳定律揭示了其操控光路的基本规律，等离激元效应提供了实现相位调制的物理机制。
2. **数学原理**：从费马原理推导了广义斯涅耳定律，并给出了局域表面等离激元共振频率的解析公式。
3. **代码实践**：用Python模拟了纳米棒超表面的相位设计，并验证了光束偏折效果。
4. **应用前景**：从平面透镜到全息显示，超表面正在重新定义光学器件的形态。

随着纳米加工技术的进步（如电子束光刻），超表面的性能不断提升，有望在未来取代许多传统光学元件，开启“平面光学”的新纪元。
