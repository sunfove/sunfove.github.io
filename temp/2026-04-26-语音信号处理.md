# 语音信号处理：从STFT到MFCC与LPC的深度解析

## 1. 引言

语音是人类最自然的交互方式，而语音信号处理则是将这种模拟信号转化为计算机可理解、可分析的数字信息的关键技术。从Siri的语音识别到实时降噪耳机，从语音合成到情感分析，语音信号处理无处不在。然而，原始语音信号包含大量冗余信息和噪声，如何提取出具有判别性的特征成为核心挑战。本文将深入剖析三个经典而强大的工具：短时傅里叶变换（STFT）、梅尔频率倒谱系数（MFCC）和线性预测编码（LPC），从数学原理到代码实现，带你掌握语音特征提取的精髓。

## 2. 核心概念

### 2.1 短时傅里叶变换 (STFT)

语音信号是非平稳的，其统计特性随时间变化。直接对整个信号使用傅里叶变换会丢失时间信息。STFT的核心思想是：**假设信号在短时间内是平稳的**，通过滑动窗口将信号分割成多个短时间段，对每个窗口进行傅里叶变换，从而得到时频联合表示。

数学上，离散STFT定义为：
$$X(m, k) = \sum_{n=0}^{N-1} x(n) w(n - mH) e^{-j\frac{2\pi k n}{N}}$$

其中，$x(n)$是输入信号，$w(n)$是窗口函数（如汉明窗），$m$是帧索引，$H$是帧移，$N$是FFT长度，$k$是频率索引。

### 2.2 梅尔频率倒谱系数 (MFCC)

人耳对频率的感知是非线性的：对低频变化更敏感，对高频变化相对迟钝。梅尔刻度正是模拟了这一特性。MFCC通过将线性频率映射到梅尔频率，再进行倒谱分析，得到一组紧凑的特征系数，广泛应用于语音识别。

梅尔频率与线性频率的转换公式：
$$f_{mel} = 2595 \cdot \log_{10}(1 + \frac{f}{700})$$

### 2.3 线性预测编码 (LPC)

LPC基于**声道模型**：语音信号可以看作由声门激励信号通过一个全极点滤波器产生。LPC通过线性预测误差最小化，估计出声道滤波器的参数（即预测系数），从而用少量参数表示语音的频谱包络。

LPC的核心是求解Yule-Walker方程，得到$p$阶线性预测系数$a_1, a_2, ..., a_p$。

## 3. 数学原理

### 3.1 STFT的数学推导

STFT可以看作是在时频平面上的“窗口化傅里叶变换”。其能量谱（频谱图）为：
$$S(m, k) = |X(m, k)|^2$$

为了理解STFT的时频分辨率，考虑窗口长度$N$与频率分辨率$\Delta f$和时间分辨率$\Delta t$的关系：
$$\Delta f \propto \frac{1}{N}, \quad \Delta t \propto N$$

这体现了**海森堡不确定性原理**：我们无法同时获得高时间分辨率和高频率分辨率。选择窗口长度需要权衡。

### 3.2 MFCC的完整计算流程

1. **预加重**：补偿高频分量，增强频谱平坦度
   $$y(n) = x(n) - \alpha x(n-1), \quad \alpha \approx 0.97$$

2. **分帧加窗**：应用STFT得到频谱$X(m, k)$

3. **计算功率谱**：$P(m, k) = |X(m, k)|^2$

4. **梅尔滤波器组**：将线性频率映射到梅尔刻度，使用三角滤波器
   $$H_m(k) = \begin{cases} 
   0 & k < f(m-1) \\
   \frac{k - f(m-1)}{f(m) - f(m-1)} & f(m-1) \leq k \leq f(m) \\
   \frac{f(m+1) - k}{f(m+1) - f(m)} & f(m) \leq k \leq f(m+1) \\
   0 & k > f(m+1)
   \end{cases}$$

5. **对数能量**：$E(m) = \ln(\sum_k P(m, k) H_m(k))$

6. **离散余弦变换 (DCT)**：得到倒谱系数
   $$c_n = \sum_{m=0}^{M-1} E(m) \cos(\frac{\pi n (m+0.5)}{M})$$

### 3.3 LPC的数学基础

LPC假设当前采样点$x(n)$可由前$p$个样本线性预测：
$$\hat{x}(n) = \sum_{i=1}^p a_i x(n-i)$$

预测误差为$e(n) = x(n) - \hat{x}(n)$。通过最小化均方误差：
$$E = \sum_n e^2(n)$$

得到Yule-Walker方程：
$$\sum_{i=1}^p a_i R(i-j) = -R(j), \quad j=1,2,...,p$$

其中$R(k)$是信号的自相关函数。求解此方程（常用Levinson-Durbin递推）得到预测系数。

## 4. 代码实现

下面提供完整的Python代码，实现STFT、MFCC和LPC特征提取：

```python
import numpy as np
import librosa
import scipy.signal as signal
import matplotlib.pyplot as plt

# 1. STFT实现
def compute_stft(x, n_fft=512, hop_length=256, win_length=512):
    """
    计算短时傅里叶变换
    :param x: 输入信号 (1D numpy array)
    :param n_fft: FFT长度
    :param hop_length: 帧移
    :param win_length: 窗口长度
    :return: 频谱矩阵 (频率轴 x 时间轴)
    """
    # 创建汉明窗
    window = np.hamming(win_length)
    
    # 计算帧数
    num_frames = 1 + (len(x) - win_length) // hop_length
    
    # 初始化频谱矩阵
    spectrogram = np.zeros((n_fft // 2 + 1, num_frames), dtype=np.complex64)
    
    for t in range(num_frames):
        # 提取当前帧
        start = t * hop_length
        frame = x[start:start + win_length] * window
        
        # 执行FFT
        fft_frame = np.fft.fft(frame, n=n_fft)
        spectrogram[:, t] = fft_frame[:n_fft // 2 + 1]
    
    return spectrogram

# 2. MFCC实现（简化版）
def compute_mfcc(x, sr=16000, n_mfcc=13, n_fft=512, hop_length=256, n_mels=40):
    """
    计算梅尔频率倒谱系数
    :param x: 输入信号
    :param sr: 采样率
    :param n_mfcc: 返回的MFCC系数数量
    :return: MFCC矩阵 (n_mfcc x 时间帧)
    """
    # 使用librosa的Mel滤波器组
    mel_basis = librosa.filters.mel(sr=sr, n_fft=n_fft, n_mels=n_mels)
    
    # 计算STFT幅度谱
    stft = compute_stft(x, n_fft, hop_length)
    power_spectrum = np.abs(stft) ** 2
    
    # 应用Mel滤波器
    mel_spectrum = np.dot(mel_basis, power_spectrum)
    
    # 对数变换
    log_mel = np.log(mel_spectrum + 1e-10)
    
    # DCT变换
    mfcc = np.dot(librosa.filters.dct(n_mfcc, log_mel.shape[0]), log_mel)
    
    return mfcc

# 3. LPC实现
def compute_lpc(x, order=12):
    """
    计算线性预测系数
    :param x: 输入信号（一帧）
    :param order: LPC阶数
    :return: LPC系数 (包含a0=1)
    """
    # 计算自相关
    n = len(x)
    R = np.correlate(x, x, mode='full')
    R = R[n-1:]  # 取非负延迟
    
    # Levinson-Durbin递推
    a = np.zeros(order + 1)
    a[0] = 1.0
    E = R[0]
    
    for i in range(1, order + 1):
        # 计算反射系数
        k = -np.dot(a[:i], R[1:i+1][::-1]) / E
        
        # 更新预测系数
        a_new = np.zeros(i + 1)
        a_new[0] = a[0]
        a_new[1:i+1] = a[1:i+1] + k * a[1:i+1][::-1]
        a_new[i] = k
        
        a = a_new
        E = E * (1 - k**2)
    
    return a

# 示例使用
if __name__ == "__main__":
    # 生成测试信号（语音模拟）
    t = np.linspace(0, 1, 16000)
    x = np.sin(2 * np.pi * 440 * t) + 0.5 * np.sin(2 * np.pi * 880 * t)
    
    # 计算STFT
    stft_result = compute_stft(x)
    print(f"STFT形状: {stft_result.shape}")  # (257, 62)
    
    # 计算MFCC
    mfcc_result = compute_mfcc(x)
    print(f"MFCC形状: {mfcc_result.shape}")  # (13, 62)
    
    # 计算LPC（取第一帧）
    first_frame = x[:512] * np.hamming(512)
    lpc_coeffs = compute_lpc(first_frame, order=12)
    print(f"LPC系数: {lpc_coeffs}")
    
    # 可视化STFT频谱图
    plt.figure(figsize=(10, 4))
    plt.imshow(np.abs(stft_result), aspect='auto', origin='lower', 
               extent=[0, len(x)/16000, 0, 8000])
    plt.colorbar(label='Magnitude')
    plt.title('STFT Spectrogram')
    plt.xlabel('Time (s)')
    plt.ylabel('Frequency (Hz)')
    plt.tight_layout()
    plt.show()
```

## 5. 应用场景

### 5.1 语音识别系统

MFCC是目前最主流的语音特征之一。在传统GMM-HMM语音识别系统中，MFCC作为输入特征，配合差分系数（ΔMFCC、ΔΔMFCC）能有效捕捉语音的动态特性。现代深度学习系统（如DeepSpeech、Wav2Vec）虽然可以直接处理原始波形，但MFCC仍然作为重要的特征工程手段，在数据量有限时能显著提升性能。

### 5.2 语音编码与压缩

LPC在语音编码领域有经典应用——**线性预测编码语音合成**。以GSM语音编码标准为例，LPC系数仅需传输少量参数，接收端通过激励信号和LPC滤波器即可重构语音。这实现了极高的压缩比（约8:1），同时保持了可懂度。此外，LPC也被用于语音增强，通过估计噪声环境下的LPC残差进行降噪。

## 6. 总结

本文深入探讨了语音信号处理中的三个核心技术：

1. **STFT**：提供了时频联合分析的基础框架，是理解语音频谱演变的窗口
2. **MFCC**：模拟人耳听觉特性，将频谱压缩为紧凑的倒谱特征，成为语音识别的“黄金标准”
3. **LPC**：基于声道模型，用线性预测参数高效表示语音频谱包络，在编码和增强中发挥重要作用

从数学推导到代码实现，我们看到了这些技术的内在联系：STFT提供时频表示，MFCC在此基础上进行感知变换，而LPC则从时域角度建模声道系统。理解这些原理，将帮助你在语音信号处理的广阔领域中游刃有余地选择和应用合适的工具。
